01
GP(글라이드패스) 개요
항공기를 정확한 강하각으로 수직 유도
📡 주파수

329~335 MHz (UHF)
LOC 주파수와 쌍으로 배정

📐 활공각

보통 3.0° (2.5°~3.5°)
패스폭: 1.4° (±0.7°)

📍 위치

접지점 측방 약 120m
활주로에 평행하게 설치

📌 GP 형식 3가지
  • Null Reference (NR): 2개 안테나, 지면 평탄 필요 — 국내 대부분
  • Sideband Reference (SR): 3개 안테나, 지면 평탄 필요
  • Capture Effect (CE): 3개 안테나, 지면 불평탄 OK
02
Null Reference GP 동작 원리
이미지 안테나를 이용해 활공각에 Null을 만든다

Null Reference GP는 Upper 안테나(측파대)와 Lower 안테나(반송파)로 구성된다. 지면 반사를 이용해 이미지 안테나를 만들고, 합성 패턴의 Null이 활공각(예: 3°)에 오도록 설계한다.

Null Reference GP — 이미지 안테나 원리
💡 Null이 활공각에 형성되는 이유
Upper 안테나의 SBO 신호는 Lower 안테나의 CSB와 역위상으로 공급된다. Upper 안테나+이미지 쌍(SOP)의 방사패턴 Null이 활공각 방향에 오도록 높이를 설계한다. 활공각에서 SBO=0 → DDM=0 → 온 패스!
03
GP 안테나 높이비
Lower:Upper = 1:2 — 3°에 Null을 만드는 수학
Null Reference GP 안테나 높이
h_U = 2 × h_L
Lower 안테나 높이 계산
h_L = λ / (4 × sinφ)
λ = GP 주파수 파장 (332MHz 기준 ≈ 0.904m)
φ = 활공각 (예: 3°)
예: h_L = 0.904 / (4 × sin3°) = 0.904 / 0.209 ≈ 4.33m
h_U = 2 × 4.33 ≈ 8.66m
⚠️ 높이 오차의 영향
Lower 안테나 높이가 변하면 SBO Null의 방향이 변한다 → 활공각이 틀어진다. 예: 높이 5cm 오차 → 패스각 수십분(') 오차 → ICAO 허용 오차 초과 가능
04
GP DDM과 패스폭
활공각 위아래에서 DDM이 변하는 방식
GP DDM 계산 (선형 구간)
DDM = 4mA × cos(h_L × sinθ)
m = 변조지수 (GP: 0.4)
A 비율 = E_ss/E_cs = (패스폭/2 + 활공각) / 10
활공각(DDM=0) 위아래로 선형 증가 → ±0.7° 에서 DDM=0.175
높이각위치DDM항공기 지시
3.0° (활공각)온 패스0.000중앙
3.7° (+0.7°)패스 위0.175Fly Down (최대편향)
2.3° (-0.7°)패스 아래0.175Fly Up (최대편향)
>5° 이상클리어런스Flag신호 불신뢰
05
Capture Effect GP
지면이 불평탄한 공항의 해결책

지면이 불평탄하거나 장애물이 있어 Null Reference GP를 쓸 수 없는 공항에 사용한다.

Null Reference 한계
  • 지면 평탄도가 매우 중요
  • 지면 불평탄 → 이미지 안테나 위상 오차 → 패스각 틀어짐
  • 산악 공항·해안 공항 적용 어려움
Capture Effect 해결 방법
  • 3개 안테나: Course·Clearance·Reference
  • 강한 Course 신호가 약한 클리어런스 신호를 '포획'
  • 지면 반사에 덜 의존 → 불평탄 지면 OK
  • 지면 조건 불리한 공항에 적합
06
근접효과 (Proximity Effect)
GP 주변 구조물·지형이 패스를 왜곡한다

GP 안테나 주변의 건물, 차량, 지형 변화 등이 의도되지 않은 반사파를 만들어 DDM 측정값을 왜곡시킨다.

⚠️ 근접효과 발생 조건
  • GP 전방 지면의 반사 물체
  • 안테나 간 거리의 100배 이내 (P < 100a)
  • 이 거리에서는 원거리 근사식(asinθ)이 부정확
  • 측정 DDM 오차 발생
📌 실무 대응
  • Critical Area: 항공기 이동 금지 구역
  • Sensitive Area: 지상 차량 이동 제한
  • GP 전방 지면 평탄화 유지
  • 신설 구조물 영향 평가 필수
07
연습문제
버튼을 눌러 정답 확인
📝 연습문제
Q1. GP 332MHz에서 활공각 3.0°인 Null Reference GP의 Lower 안테나 높이를 계산하라. (c=3×10⁸m/s)
λ = 3×10⁸ / 332×10⁶ ≈ 0.904m
h_L = λ/(4×sinφ) = 0.904/(4×sin3°) = 0.904/(4×0.05234) = 0.904/0.2094 ≈ 4.32m
Upper 안테나: h_U = 2×4.32 ≈ 8.64m
Q2. GP 측정 시 항공기나 측정 장비를 GP 안테나 배열 근처(100a 이내)에 두면 안 되는 이유는?
두 안테나까지의 실제 경로차가 원거리 근사식(asinθ)과 달라지는 근접효과(Proximity Effect) 때문이다. 가까운 위치에서는 두 안테나까지의 경로가 평행하지 않아 위상 오차가 발생하고, 측정된 DDM 값이 실제 원거리에서의 값과 다르게 나타난다.
