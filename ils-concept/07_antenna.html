01
기본 안테나 쌍 (Basic Antenna Pair)
두 안테나에 신호를 공급하면 방사패턴이 만들어진다
⚡ 핵심 질문
단일 안테나는 모든 방향으로 균일하게 방사한다. 그런데 두 안테나를 나란히 놓고 신호를 공급하면 왜 특정 방향에는 강하고, 다른 방향에는 0(Null)이 될까?

두 안테나에서 방사된 신호는 공간의 각 지점에서 합성된다. 합성 결과는 두 신호의 위상 관계에 따라 결정된다.

합성 전계강도 계산 (일반식)
E_t = E₁∠β₁ + E₂∠β₂
β₁ = φ₁ + a·sinθ    (안테나 1 신호의 P점 위상)
β₂ = φ₂ − a·sinθ    (안테나 2 신호의 P점 위상)
a = 안테나간 반거리 (전기각), θ = 관측 방향각
02
동위상 쌍 — SIP (Specific In-phase Pair)
같은 위상을 공급하면 정면이 최대, CSB에 사용

두 안테나에 동일한 위상(φ₁=φ₂=0°)의 전류를 공급한다.

SIP 합성 전계강도
E_SIP = 2I · cos(a · sinθ)
θ=0° (정면): cos(0)=1 → E=2I (최대)
a·sinθ=90°일 때: cos(90°)=0 → Null
Null 방향: θ = arcsin(90°/a)
SIP 방사패턴 — 정면(0°) 최대
03
역위상 쌍 — SOP (Specific Out-of-phase Pair)
반대 위상을 공급하면 정면 Null, SBO에 사용

한 안테나에 0°, 다른 안테나에 180° 위상의 전류를 공급한다.

SOP 합성 전계강도
E_SOP = 2I · sin(a · sinθ)
θ=0° (정면): sin(0)=0 → 항상 Null!
a·sinθ=90°일 때: sin(90°)=1 → 최대
좌측 로브와 우측 로브는 180° 역위상 관계 (Bi-phase 패턴)
SOP 방사패턴 — 정면 Null, 좌우 역위상
⚡ SOP의 Bi-phase 특성이 핵심!
SOP 패턴에서 위쪽 로브와 아래쪽 로브(또는 왼쪽·오른쪽)는 180° 역위상이다. 이것이 CSB와 합성될 때 한쪽은 90Hz 우세, 반대쪽은 150Hz 우세가 되는 원리다.
04
다중 안테나 배열
여러 쌍을 조합하면 더 정밀한 패턴을 만든다

여러 안테나 쌍을 조합하면 개별 패턴의 합성으로 더 복잡하고 정밀한 방사패턴을 형성할 수 있다.

📌 다중 배열의 합성 패턴 계산
  • 각 안테나 쌍의 패턴을 개별 계산
  • 관측 방향 θ에서 각 쌍의 합성 페이서를 구함
  • 모든 쌍의 페이서를 벡터합 → 최종 패턴
  • 페이서의 합이 0인 방향 = Null
LOC 배열 (8~16 소자)
  • 중앙 소자: CSB 강함 (SIP)
  • 외곽 소자: SBO 강함 (SOP)
  • 정밀한 코스폭·클리어런스 패턴
  • 분배망(Distribution Unit)으로 위상·진폭 배분
GP 배열 (2~3 소자)
  • Lower·Upper 안테나 (Null Reference)
  • 또는 Lower·Upper·Reference (Sideband Reference)
  • 지면 반사 활용 (이미지 안테나)
  • 높이비 = 1:2로 3° Null 형성
05
이미지 안테나 (Image Antenna)
지면을 거울로 사용해 가상의 안테나를 만든다

지면 위에 설치된 안테나가 방사하면, 지면이 전파를 반사한다. 이 반사파는 마치 지면 아래 대칭 위치에 가상의 안테나가 있는 것처럼 분석할 수 있다.

수직편파의 이미지 안테나

지면 반사 시 위상 보존
→ 이미지 안테나 전류 = 실제 안테나와 동위상
→ 실제 안테나+이미지 안테나 = SIP 구성

수평편파의 이미지 안테나

지면 반사 시 위상 180° 반전
→ 이미지 안테나 전류 = 실제 안테나와 역위상
→ 실제 안테나+이미지 안테나 = SOP 구성

💡 GP Null Reference의 원리
GP는 수직편파를 사용한다. Lower 안테나의 이미지 안테나는 지면 아래에 동위상으로 존재한다. Lower 실제+이미지 = SIP 구성 → 반송파 패턴
Upper 안테나의 측파대 신호는 Lower 반송파와 역위상으로 공급 = SOP 구성 → 활공각에 Null 형성
06
연습문제
버튼을 눌러 정답 확인
📝 연습문제
Q1. a=90°인 SIP 안테나 쌍에서 θ=30° 방향의 합성 전계강도를 기준값(θ=0°)의 비율로 구하라.
E_SIP(θ=30°) = 2I·cos(90°×sin30°) = 2I·cos(90°×0.5) = 2I·cos(45°) = 2I×0.707 = 1.414I
기준값(θ=0°) = 2I·cos(0) = 2I
비율 = 1.414I / 2I = 70.7%
Q2. SOP 방사패턴이 Bi-phase(양상) 패턴인 이유를 설명하라.
SOP 수식 E_SOP = 2I·sin(a·sinθ)에서, θ>0° 방향에서는 sin이 양수이므로 합성 전계는 양의 위상을 갖는다. 그러나 θ<0° (반대 방향)에서는 sin이 음수가 되어 합성 전계는 음의 값, 즉 180° 반전 위상이 된다. 따라서 SOP 패턴의 두 로브는 항상 서로 역위상 — Bi-phase 패턴이다.
