01
모니터 시스템의 역할
신호 이상 발생 시 항공기에 잘못된 정보가 전달되기 전에 차단
⚠️ 왜 모니터가 필수인가?
ILS 신호 이상이 발생해도 항공기 승무원이나 관제사가 즉시 알 수 없다. 모니터 시스템이 자동으로 감시하다가 이상을 감지하면 즉시 경보·절체해야만 CAT II·III 정밀 접근의 안전이 보장된다.
감시 대상
  • 코스/패스 정렬 오차
  • DDM (변조도 차)
  • 전계강도 (신호 세기)
  • 변조도 (m%)
감시 방법
  • Field Detector (방사 공간 측정)
  • 자체 모니터 (Integral Monitor)
  • 원격 모니터 (Remote Monitor)
이상 시 동작
  • 경보 (Warning) 발생
  • 알람 (Alarm) 발생
  • Tx 절체 (Tx1↔Tx2)
  • 최종: 방사 중단
02
모니터 구성 — NF Monitor와 FF Monitor
근접 모니터와 원거리 모니터의 역할 분담
NF Monitor (Near Field)

안테나 근처에 설치된 Field Detector에서 신호 샘플링

  • 빠른 응답 속도
  • 실제 방사 신호를 직접 측정
  • DDM·변조도·전계강도 감시
  • GP, LOC 모두 사용
FF Monitor (Far Field, LOC 전용)

활주로 방향 원거리에 설치된 Field Detector

  • 실제 코스 정렬을 원거리에서 검증
  • NF Monitor가 검출 못하는 미세 정렬 오차 감지
  • LOC만 적용 (GP는 지형상 어려움)
  • CAT II·III에서 특히 중요
03
GP 모니터링 (CH15)
활공각·패스폭·변조도를 감시하는 원리

GP NF Monitor는 Field Detector에서 측정한 DDM을 이용해 패스폭 변화를 감시한다.

GP 모니터 DDM 목표값 (A 비율)
A = (φ + 0.1) / 10
φ = 활공각 (°)
예: φ=3.0° → A = (3.0+0.1)/10 = 0.31
모니터 출력 DDM = 2m×(Ess/Ecs) = 2×0.4×0.31 = 0.248 (정상)
측정값이 이 값보다 크게 변하면 패스폭 이상으로 경보
모니터 항목이상 조건동작
패스각 (DDM=0 위치)0.075° 이상 이탈경보 → 알람 → 절체
패스폭0.175 DDM 위치 변화경보 → 알람
전계강도규정값 -3dB 이하Flag 알람
변조도규정값 ±20% 이상경보
04
LOC 모니터링 (CH18)
코스 정렬·코스폭·전계강도를 이중으로 감시

LOC는 NF Monitor와 FF Monitor를 함께 운용하여 이중 감시 체계를 구성한다.

1
NF Monitor (안테나 근처)

Field Detector → DDM 측정 → 코스폭·변조도 감시. 빠른 응답으로 이상 즉시 검출.

2
FF Monitor (원거리)

활주로 방향 원거리 Field Detector → 실제 코스 정렬 감시. NF만으로는 검출 어려운 미세 정렬 오차 감지.

3
이중 판정

NF·FF 중 하나라도 허용 범위 초과 → 경보. 두 모니터 모두 이상 → 알람 → Tx 절체.

05
알람 종류와 절체 동작
Warning → Alarm → 절체 → 방사 중단 단계
단계조건동작의미
Warning파라미터 허용범위 접근운용자 경보만주의 요망, 아직 정상 운용
Alarm파라미터 허용범위 초과Tx1→Tx2 자동 절체예비 송신기로 전환
Alarm (절체 후)Tx2도 이상방사 중단 (Shutdown)Flag 발생, 항공기에 신호 불신뢰 알림
Flag전계강도 미달항공기 계기에 Flag 표시조종사에게 ILS 신뢰 불가 통보
💡 운용자 대응 절차 (기본)
  1. 어느 모니터(NF/FF)에서 알람이 발생했는지 확인
  2. 어떤 파라미터(DDM/변조도/전계강도)의 이상인지 확인
  3. 절체 후 Tx2 동작 정상 여부 확인
  4. 관제탑에 ILS 상태 통보
  5. 원인 분석 후 정비 조치
06
연습문제
버튼을 눌러 정답 확인
📝 연습문제
Q1. LOC에서 FF Monitor를 NF Monitor와 별도로 운용하는 이유는?
NF Monitor는 안테나 근처에서 측정하므로 원거리(실제 접근 구간)에서의 코스 정렬 오차를 정확히 검출하기 어렵다. FF Monitor는 실제 항공기가 접근하는 원거리 방향에서 코스 정렬을 직접 측정하여 NF에서 검출하지 못하는 미세한 정렬 오차도 감지한다. 두 모니터의 상호 보완으로 더 높은 신뢰성을 확보한다.
Q2. GP NF Monitor 알람 발생 시 운용자의 확인 우선순위 3단계를 설명하라.
1단계: 어떤 모니터 채널에서 알람인지 확인 (NF Monitor 어느 포트)
2단계: 어떤 파라미터 이상인지 확인 (DDM/전계강도/변조도 중 무엇이 허용 범위 초과)
3단계: Tx 절체 후 Tx2 동작 정상 확인 및 관제탑 통보
이후 원인 분석(케이블·안테나·송신기·분배망 순)하여 정비 조치
