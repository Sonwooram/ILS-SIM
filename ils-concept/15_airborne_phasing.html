01
왜 에어본 페이징을 하는가?
운용개시 검사의 핵심 절차 — 신호 위상이 맞아야 정확한 활공로가 형성됨
⚡ 이 강의의 목표
운용개시 비행검사에서 수행되는 에어본 페이징 2가지 절차(Quadrature Phasing, Dephasing Check)를 정확히 이해하고 단계별로 따라할 수 있게 됩니다.
"복잡한 절차"가 아니라 "신호를 단계별로 하나씩 맞춰가는 과정"이라는 본질을 명확히 합니다.

1-1. 운용개시 비행검사 항목 中 페이징 관련 부분

한국 항공법 GP 비행검사 표(부록)를 보면, 운용개시 검사에서 위상 관련 항목이 2개 있습니다:

번호 항목 허용범위
9 위상 (Phasing) 최초 설정되는 변조 균형 값은 가능한 한 0μA 이내로 유지되어야 한다
10 위상조정 후 확인검사
(Phasing and Airborne Phase Verification)		 운용개시검사 시 국토교통부 장관이 별도로 정하는 비행검사 절차에 따라 장비의 정상 작동상태를 확인하여야 한다
📌 두 항목의 의미
9번 (Phasing): 변조 균형이 0μA 이내가 되도록 최초 위상 설정을 한다 → 보통 Quadrature Phasing으로 지상에서 수행
10번 (Phasing Verification): 그렇게 설정한 위상이 비행검사 결과 실제로 정상 작동하는지 확인 → Dephasing Check로 검증

즉 9번이 "맞추는 절차", 10번이 "맞아 있는지 확인하는 절차"입니다. 두 절차가 한 쌍을 이룹니다.

1-2. 위상이 왜 그렇게 중요할까?

GP 시설의 활공로는 공간 합성으로 형성됩니다. CSB(반송파)와 SBO(측파대)가 각각 다른 안테나에서 방사되어 공간에서 합쳐지죠. 이때 두 신호의 RF 위상 관계가 정확해야:

  • 활공각에서 DDM = 0 (정확한 활공로 형성)
  • 활공각 위는 90Hz 우세, 아래는 150Hz 우세 (대칭 분포)
  • 패스폭이 1.4° (±0.7°) 좌우 대칭

만약 위상이 어긋나면 → DDM이 0이 되는 지점이 활공각이 아닌 곳에 형성되고, 패스폭이 비대칭이 되고, 결국 항공기가 잘못된 경로로 안내됩니다.

1-3. 페이징은 운용개시 + 비정기 시에만

한국 항공법 비행검사 표를 다시 보면, 위상 관련 검사 9·10번은 모두 운용개시에만 ●표시가 있습니다. 정기검사에서는 하지 않아요. 이유는:

  • 일단 정확히 위상 조정을 해두면 오랫동안 안정적으로 유지됨 (RF 케이블·분배망의 위상 특성은 거의 변하지 않음)
  • 정기검사에서는 대칭성·DDM 균형 등 다른 항목으로 위상 정상성이 간접적으로 확인됨
  • 비행검사는 비용·시간이 큰 작업이라 매번 위상까지 점검할 필요 없음
⚠️ 페이징 재실시가 필요한 경우
• 운용개시 검사 (필수)
• 장비 교체: 송신기·분배망·안테나 교체 후
• 케이블 교체: 메인 피더 또는 분배망 케이블 교체 후
• 대규모 시설 보수 후
• 비행검사에서 위상 이상 발견 시 (대칭성·구조 이상)
02
위상이 어긋나면? — cos θ의 의미 ★
패스폭이 어떻게 변하는지 수식으로 정확히 이해

2-1. 핵심 공식 — SBO Effectiveness

SBO와 CSB의 위상 관계는 코사인 함수로 표현됩니다. 두 신호가 정확히 90° 위상 관계일 때만 DDM이 제대로 형성되는데, 위상이 어긋나면 DDM에 기여하는 비율이 cos θ로 줄어듭니다.

FORMULA — 교재 19장
SBO Effectiveness = cos(Misphasing ∠) = cos θ
Path Width = Width at Optimum Phasing / cos θ
θ: SBO 신호의 위상 오차 (0°이면 완벽, 90°이면 SBO가 전혀 작동 안 함)
→ 위상이 어긋날수록 SBO가 DDM에 기여하는 양이 줄어들어 패스폭이 넓어집니다.

2-2. 표로 보는 위상 오차 → 패스폭 변화

운용개시 기준 패스폭 0.7° (1.4° 전체 폭의 절반)을 기준으로:

SBO 위상 오차 cos θ (Effectiveness) 예상 패스폭 판정
0° (Optimum)1.0000.70°정상 ✓
±10°0.9850.71°정상 ✓
±20°0.9400.74°허용
±30°0.8660.81°경계
±45°0.7070.99°불량
±60°0.5001.40°심각
±90°0.000∞ (무한대)완전 무효!
🚨 90° 위상 오차의 의미
SBO가 CSB와 정확히 90° 어긋나면 모든 높이각에서 DDM = 0이 됩니다. 즉 어느 각도든 활공로처럼 인식되어 항공기가 정상 활공각을 찾을 수 없어요. 활공로가 형성되지 않는 위험한 상태입니다.

역설적으로 이 원리가 Quadrature Phasing 점검의 핵심이에요. 일부러 90° 케이블을 끼워 "위상 90°" 상태를 만들고, 이때 DDM이 어떻게 변하는지 보고 정상 여부 판단.

2-3. 시각화 — Optimum vs Advanced 30° vs Retarded 30°

SBO 위상 오차에 따른 페이서 합성 (교재 19장 그림)
💡 핵심 인사이트 — 대칭성이 정상 판단 기준
Initial Phasing이 0°(완벽)이라면: SBO를 +30° 앞서게 해도 / −30° 늦게 해도 패스폭이 똑같이 0.81°로 넓어집니다.

이게 바로 Dephasing Check의 원리예요. 양쪽 방향으로 같은 양만큼 위상을 일부러 흐트리고, 두 패스폭이 같으면 시스템 위상이 정상. 다르면 정상이 아니라는 것.
cos 함수는 짝수 함수(cos θ = cos −θ) — 이 성질을 이용한 검사법입니다.

03
2가지 절차 한눈에 보기 ★
먼저 두 절차의 역할과 차이를 명확히 합니다

운용개시 검사에서 수행되는 페이징은 2단계 짝꿍 절차예요. 먼저 지상에서 정확히 맞추고(①), 비행검사로 검증하는(②) 구조입니다.

① Quadrature Phasing 지상 작업

  • 목적: 위상을 맞추는 절차
  • 장소: GP 시설 지상 (송신기실)
  • 도구: 90° 케이블, 더미로드, PIR
  • 원리: SBO에 일부러 90° 케이블 삽입 → 정상이면 DDM이 CSB Only와 같아야 함
  • 출력: 송신기의 SBO 위상 조정기 설정값
  • 소요 시간: 1~3시간 (시설 차단 필요)
  • 관련 항공법: 비행검사 9번 항목 (Phasing)
→

② Dephasing Check 비행검사 중

  • 목적: 맞춰진 위상이 실제로 정상인지 확인하는 절차
  • 장소: 비행기 + 안테나 마스트 (현장 협업)
  • 도구: Bullet/Barrel 동축 아답터, GP Phasing Calculator
  • 원리: 상단/하단 안테나에 각각 30° 위상 지연 삽입 → 두 패스폭이 같으면 OK
  • 출력: 위상 오차량 → 필요시 SBO 위상 보정
  • 소요 시간: 비행검사 시간에 추가 30분~1시간
  • 관련 항공법: 비행검사 10번 항목 (Airborne Phase Verification)

3-1. 왜 2가지가 필요한가?

🎯Quadrature만 하면 안 되나?

지상에서만 측정하면 근거리 효과(Near Field)가 포함되어 실제 비행 환경과 다를 수 있어요. 특히 GP는 지면 반사를 활용하므로 원거리에서 측정해야 정확합니다.
→ 지상에서 1차 설정 후 비행으로 검증 필요.

✈️Dephasing만 하면 안 되나?

Dephasing은 위상 오차를 측정만 하는 절차예요. 직접 조정하려면 송신기에서 SBO 위상기를 만져야 하는데, 그 1차 설정값을 정확히 만들어야 하니까 지상 Quadrature가 먼저 필요.
→ "맞춤(지상)" + "검증(비행)" 한 쌍.

3-2. 두 절차의 핵심 원리 요약

두 절차의 원리 비교
🎓 두 절차의 본질
두 절차 모두 결국 "의도적으로 위상을 흐트려서, 그 결과가 예측 가능하면 정상"이라는 원리를 사용합니다.
• Quadrature: 90° 흐트림 → SBO 기여 0이 예상 → 실제로 0이면 OK
• Dephasing: ±30° 흐트림 → 양쪽 같은 양 → 실제로 같으면 OK

어느 쪽이든 cos 함수의 대칭성을 이용한 검사법이에요.
04
절차 ① Quadrature Phasing (90° 케이블법)
지상에서 송신기 SBO 위상을 직접 조정

4-1. 절차 개요 — 무엇을 단계별로 맞추는가

이 절차는 송신기의 SBO 위상 조정기를 정확한 값으로 설정하는 게 목적이에요. 단계별로 뭘 맞추는지 한 줄 정리:

1
기준값 측정
CSB만 방사 시 DDM을 측정 → 이게 "위상이 의미 없는 상태"의 기준
→
2
90° 인공 흐트림
SBO에 90° 케이블 삽입 → 정상이면 SBO 기여가 0이 되어 CSB Only와 같아야 함
→
3
SBO 위상 조정
DDM이 기준값과 일치할 때까지 송신기 SBO 위상 조정기를 돌림
→
4
정상화 확인
90° 케이블 제거 → 원래 상태로 복원 → 150Hz 우세 정상 확인

4-2. 상세 절차 (교재 14장)

NRGP 시설 기준 절차를 자세히 보겠습니다. CEGP도 유사하지만 측파대가 3개 안테나로 분배되므로 약간의 차이가 있어요.

1
준비
PIR을 미들마커 지점, 활공각 아래쪽에 설치
• 활공각 3.0°라면 약 1.5° 위치에 PIR (Portable ILS Receiver) 설치
• 이 지점에서는 정상 시 150Hz가 우세한 DDM이 측정되어야 함
• 안테나로부터 충분한 거리 확보 (보통 미들마커 지점)
2
기준값 측정
CSB Only 상태로 방사 → 기준 DDM 기록
실행:
• 장비 전원 OFF
• 측파대(SBO) 안테나로 가는 전송선 분리
• 송신기의 SBO 출력 단자에 50Ω 더미로드 연결 (반사파 방지)
• 전원 ON → 송신기 동작

결과: CSB 신호만 방사됨 (SBO는 더미로드에서 흡수)
→ PIR이 표시하는 DDM 값을 정확히 기록 (이게 기준값)

왜 이게 기준이 되는가? CSB만 있으면 SBO 위상이 어떻든 상관 없는 상태. 이 DDM 값에 SBO 90° 삽입한 결과가 일치해야 정상.
3
90° 케이블 삽입
SBO 신호 경로에 90° 위상 케이블 삽입
실행:
• 장비 전원 OFF
• SBO 출력단의 더미로드 제거, 원래 SBO 전송선 다시 연결
• SBO 전송선 중간에 반송파 주파수의 90°에 해당하는 길이의 케이블 삽입

90° 케이블 길이 계산 (GP 330MHz 기준, VF=0.66 RG-214):
• λ_air = 300 / 330 = 0.909 m
• λ_cable = 0.66 × 0.909 = 0.600 m
• 90° = 1/4 λ → L = 0.600 / 4 = 15.0 cm

효과: 측파대 신호의 페이서가 시계방향으로 90° 회전 → SBO와 CSB가 정확히 90° 위상차 상태
4
조정
송신기 SBO 위상 조정기로 DDM 일치시킴
실행:
• 전원 ON → 송신기 정상 동작
• PIR의 DDM 수치 확인
• 단계 2에서 기록한 CSB Only 기준값과 일치하도록 송신기 내부의 SBO 위상 조정기를 천천히 돌림

이론적 근거: 위상이 정확히 90° 인공 흐트림된 상태에서 cos(90°+θ) ≈ 0이 되려면, 시스템의 자연 위상 오차 θ가 0이어야 함. 즉 DDM이 기준값과 같다 = 시스템 위상이 정확히 0°

주의: 위상 조정기는 매우 미세한 조작이 필요. 한 번에 5°씩 돌리지 말고 1~2° 단위로
5
정상화 확인
90° 케이블 제거 → 원상 복귀 → 정상 동작 확인
실행:
• 장비 전원 OFF
• 삽입했던 90° 케이블 제거
• SBO 전송선을 원래대로 복원
• 전원 ON

확인: PIR에 표시되는 DDM이 정상(150Hz 우세)인지 확인
→ 활공각 위쪽으로 이동하면 90Hz 우세로 바뀌는지도 확인하면 더 좋음

4-3. CEGP에서의 추가 고려사항

⚠️ CEGP의 차이점
교재 13장: "상단과 하단 안테나에서의 클리어런스 신호의 RF 위상은 코스 신호 측파대 성분의 위상이 정상적으로 조정되면 자동으로 동기됨"

즉 CEGP에서도 기본 절차는 NRGP와 같이 코스 신호의 SBO 위상을 먼저 정확히 조정합니다. 그 결과로 CLR 신호 위상도 자동 정렬돼요.
다만 CEGP는 SBO가 3개 안테나에 분배되므로 (−0.5 : 1 : −0.5), 위상이 한 곳에서 어긋나면 다른 항목들에 더 광범위한 영향을 줄 수 있어요. 그래서 운용개시 시 더 정밀한 측정이 필요합니다.

4-4. Quadrature Phasing 시 자주 발생하는 문제

🔧케이블 길이 오차

90° 케이블이 정확히 90°가 아니면 기준값 비교가 부정확해짐.
→ 사용 전 VNA로 정확한 위상 확인. ±1° 이내 정밀도 권장.

📍PIR 위치 부적절

근거리에서는 안테나 오프셋 영향으로 측정 오차 발생. 미들마커 또는 그 이상 거리 필수.
→ 활주로 시단 중앙도 좋은 위치 (활공각 위 약 50ft)

⚡전원 차단 안 함

케이블 삽입/분리 시 전원을 끄지 않으면 송신기 보호회로 동작 또는 손상 가능. 절대 ON 상태로 작업 금지.

🌡️온도 영향

새벽·낮 등 극단적인 온도차에서는 케이블 위상이 변할 수 있음.
→ 가능하면 오전 또는 안정된 온도 환경에서 작업.

5-2. 상세 절차 (교재 19장)

1
기준 측정
정상 상태에서의 패스폭 측정 (Optimum Phasing 가정)
Quadrature Phasing이 완료된 상태에서 비행검사로 패스폭 측정
• 목표값: 패스폭 0.7° (운용개시 ±0.07°)
• 측정 위치: ICAO Zone 2 또는 비행검사 표준 측정점

이 값을 W₀로 기록.
2
상단 위상 지연
상단 안테나 전송선에 30° 위상 지연 삽입
실행:
• 안테나 마스트에서 상단 안테나 전송선 분리
• 동축 아답터 (Bullet 또는 Barrel) 삽입 — 약 30° 위상 지연
• 전송선 재연결 → 시설 정상 동작 확인

Bullet/Barrel 30° 길이:
• λ_cable = 0.66 × 0.909 = 0.600 m
• 30° = (30/360) × λ_cable = 약 5.0 cm
• 실무에서는 미리 제작된 표준 위상 아답터 사용
3
비행검사 ①
비행검사 → 패스폭 W₁ 측정
• 비행검사기로 정상 비행 → 패스폭 측정
• 측정값을 W₁으로 기록

예상값 (위상 정상이라면):
• 상단이 30° 지연되면 SBO Retarded −30°와 같은 효과
• cos(−30°) = 0.866 → W₁ ≈ 0.7 / 0.866 = 0.81°
4
하단 위상 지연으로 교체
상단 Bullet 제거, 하단 안테나 전송선에 동일 Bullet 삽입
실행:
• 상단 안테나 전송선에서 Bullet 제거 → 정상 복귀
• 하단 안테나 전송선에 같은 Bullet 삽입
• 시설 정상 동작 확인

효과: 하단이 30° 지연된 것은 상대적으로 상단이 30° 앞선 것과 같은 효과 → SBO Advanced +30° (cos 함수의 대칭성)
5
비행검사 ②
비행검사 → 패스폭 W₂ 측정
• 동일한 비행 패턴으로 측정
• 측정값을 W₂로 기록

예상값 (위상 정상이라면): W₂ ≈ 0.81° (W₁과 같음)
6
평가
W₁과 W₂를 비교 → 위상 정상 여부 판정
판정 기준 (교재 19장):
• |W₁ − W₂| ≤ 0.03° ~ 0.04°: 위상 정상 ✓ → 추가 조정 불필요
• |W₁ − W₂| > 0.04°: 위상 어긋남 → 다음 단계로

어긋남이 발견되면: GP Phasing Calculator로 위상 오차량 계산 → 송신기에서 SBO 위상 보정 → 다시 Dephasing Check 반복

케이스 W₁ (상단 30° 지연) W₂ (하단 30° 지연) 차이 판정
이상적0.81°0.81°0.00°완벽 ✓
정상 범위0.80°0.82°0.02°정상 ✓
경계0.79°0.83°0.04°경계 ⚠
약간 어긋남0.78°0.86°0.08°조정 필요

📌 왜 0.04°가 기준일까?
Initial Phasing이 ±5° 이내라면 두 패스폭 차이는 0.04° 이하가 됩니다. 그 이상의 차이는 시스템 위상이 ±5° 이상 어긋났다는 의미예요. 운용개시 검사의 정확도 기준입니다.

5-4. Dephasing Check 시 주의사항

⚙️아답터 위상 정확도

Bullet/Barrel의 실제 위상이 정확히 30°가 아니면 결과 해석이 흐려짐.
→ 사용 전 VNA로 측정해서 ±1° 이내 확인. 표준 아답터 권장.

🛬동일 비행 조건

W₁과 W₂ 측정 시 같은 비행 패턴·고도·속도 유지. 다른 조건에서 측정한 값은 비교 불가.

📡시설 안정화 대기

Bullet 교체 후 시설이 안정화되는 데 5~10분. 즉시 측정하지 말고 안정 후 측정.

🔄비행시간 절약

비행기는 비싼 시간 자원. 지상 작업(아답터 교체)을 신속히 처리할 수 있도록 사전 준비·연습 필수.

GP Phasing Calculator 사용법 ★
Dephasing Check 결과로 정확한 보정량을 산출하는 도구

6-1. Calculator의 원리

GP Phasing Calculator는 위상 오차량과 패스폭의 관계 그래프입니다. 교재 19장 그림 19-22.를 참고하면 U자형(또는 V자형) 곡선이에요. 가로축은 위상 불일치 각도, 세로축은 패스폭이죠.

FORMULA — 패스폭 계산 (Calculator 곡선)
Path Width(θ) = 0.7° / cos(θ)
θ는 SBO 위상 오차(degrees). 곡선의 최저점이 θ=0°일 때 패스폭 0.7° (Optimum).
양쪽으로 갈수록 패스폭이 증가하는 대칭 곡선.

6-3. Calculator 단계별 사용법

1
데이터 입력

예제 계산:

• 또는 측파대 안테나 케이블을 10.5°에 해당하는 길이만큼 연장

10.5° 케이블 길이 (GP, VF=0.66): (10.5/360) × 0.6m ≈ 1.75 cm

→ |W₁' − W₂'| ≤ 0.04° 도달할 때까지 반복

Initial Phasing이 어긋날 때 — 비대칭 분석
왜 두 패스폭이 다르게 나오는지 표로 이해

교재 19장 표 19-1.은 Initial Phasing 상태와 Dephasing Check 결과의 관계를 명확히 보여줍니다.

7-1. 교재 19장 표 19-1 (Initial Phasing vs. Path Width)

Initial Phasing Facility Condition Resultant Phasing Path Width 분석
0° Normal 0° 0.70° 정상 기준값
0° Advanced 30° +30° 0.81° 대칭 ✓
0° Retarded 30° −30° 0.81° 대칭 ✓ — Initial이 0°이면 좌우 같음
─── Initial Phasing이 어긋난 경우 ───
−10° Normal −10° 0.71° 패스폭은 정상처럼 보이지만 위상 어긋남
−10° Advanced 30° +20° 0.75° Initial 어긋남 → 다른 패스폭
−10° Retarded 30° −40° 0.91° 차이 0.16° → Dephasing 검출됨

7-2. 핵심 통찰 — 비대칭이 곧 Initial Phasing 오차

🎯 표에서 발견되는 패턴
Initial Phasing = 0° (정상):
→ Advanced +30° = Retarded −30° → 둘 다 0.81°로 같음

Initial Phasing = −10° (어긋남):
→ Advanced +30° = 결과 +20° → 0.75°
→ Retarded −30° = 결과 −40° → 0.91°
→ 차이 0.16° 발생 → 검출!

즉 비대칭의 양이 곧 Initial Phasing 오차의 양입니다. 그래서 Calculator로 그 양을 정확히 계산할 수 있어요.

7-3. 비대칭이 발생하는 이유 — 수학적 설명

FORMULA — 두 케이스의 패스폭
W₁ = 0.7° / cos(θ_init + 30°)
W₂ = 0.7° / cos(θ_init − 30°)
θ_init = Initial Phasing 오차
θ_init = 0이면 cos(+30°) = cos(−30°)이므로 W₁ = W₂
θ_init ≠ 0이면 cos 함수의 비대칭으로 W₁ ≠ W₂

두 값의 차이가 클수록 Initial Phasing 오차가 크다는 의미.

7-4. 위상 오차 부호 판별법

Calculator로 위상 오차의 크기는 알 수 있지만, 부호(앞섬 vs 뒤짐)는 어떻게 판별할까요?

↗️W₂ > W₁ (하단 지연 시 더 넓음)

하단 30° 지연 = 상대적으로 상단 30° 앞섬
→ 이때 더 넓어진다는 건 결과 위상이 더 어긋난다는 의미
→ 원래도 SBO가 앞서 있었다는 뜻
→ SBO Advanced (앞섬) → 송신기에서 SBO 늦춤

↘️W₁ > W₂ (상단 지연 시 더 넓음)

상단 30° 지연 = SBO 30° 뒤짐
→ 이때 더 넓어진다는 건 원래도 SBO가 뒤져 있었다는 뜻
→ SBO Retarded (뒤짐) → 송신기에서 SBO 앞당김

💡 외울 필요 없는 직관
"Bullet을 어디 끼웠을 때 더 나빠지냐가, 원래 문제 방향"
더 패스폭이 넓어진 쪽 = 같은 방향으로 더 어긋난 것 = 원래도 그 방향으로 문제 있음
→ 송신기에서 반대 방향으로 보정
08
전체 운용개시 페이징 플로 ★
2가지 절차를 합쳐서 운용개시 검사 시 어떻게 진행하는가

실제 운용개시 검사 현장에서는 두 절차가 한 패키지로 진행됩니다. 전체 흐름을 단계별로 정리:

A
A단계: 지상 준비 (Quadrature Phasing 사전)
• 시설 설치·결선 완료 확인
• 변조도·균형 1차 조정
• PIR 미들마커 지점 설치
• 90° 케이블 준비 (15cm RG-214)
B
B단계: Quadrature Phasing 수행 (지상)
• CSB Only 기준 DDM 측정 → 기록
• SBO에 90° 케이블 삽입
• 송신기 SBO 위상 조정 → 기준 DDM과 일치
• 90° 케이블 제거 → 정상 동작 확인
결과: 1차 위상 설정 완료 (지상 기준)
C
C단계: 비행검사 — 기준 패스폭 측정
• 정상 상태에서 패스폭 W₀ 측정 (목표 0.7° ±0.07°)
• 활공각, 변조 균형 등 기본 항목 측정
결과: 운용개시 기준 데이터 확보
D
D단계: Dephasing Check ① (상단 30° 지연)
• 상단 안테나 전송선에 30° Bullet 삽입
• 비행검사 → 패스폭 W₁ 측정 (예상 0.81°)
E
E단계: Dephasing Check ② (하단 30° 지연)
• 상단 Bullet 제거 → 하단에 같은 Bullet 삽입
• 비행검사 → 패스폭 W₂ 측정
• |W₁ − W₂| ≤ 0.04° 이면 정상 → 종료
F
F단계: 조정 필요 시 (|W₁−W₂| > 0.04°)
• Phasing Calculator로 위상 오차량 산출
• 송신기 SBO 위상 보정
• C단계부터 다시 (보통 1~2회 반복)
G
G단계: 최종 확인 — 운용개시 승인
• 모든 항목 운용개시 허용범위 충족 확인
• 검사 보고서 작성
• 변조 균형 ≤ 0μA → 항공법 9번 항목 충족 ✓
• Dephasing 정상 → 항공법 10번 항목 충족 ✓

8-1. 소요 시간과 비용 예시

단계 소요 시간 주요 자원 비고
A·B (Quadrature)2~3 시간지상팀, PIR시설 차단 필요
C (기준 비행검사)1 시간비행기·조종사·검사관기상 영향 큼
D·E (Dephasing 비행)1~1.5 시간지상팀 + 비행기 대기Bullet 교체 + 비행 반복
F (조정 + 재검증)+1~2 시간반복1~2회 반복 일반적
합계5~7 시간—당일 완료 가능
⚠️ 비행기 시간 비용
비행검사기 시간은 매우 비싸요 (시간당 수백만원). 그래서 Quadrature Phasing을 지상에서 정확히 해놓는 것이 매우 중요합니다. 지상 작업이 정확할수록 비행 반복 횟수가 줄어듭니다.
이상적인 경우: Quadrature 완벽 → Dephasing 1회만 → 종료 (비행 2~3회)
안 좋은 경우: Quadrature 부정확 → Dephasing → 보정 → Dephasing 재실시 → ... (비행 5~6회)

9-1. 교재 19장 예제 풀이

📘 교재 예제 2 (CH19)
비행검사 수행 중 상단 안테나 전송선에 동축 아답터를 삽입함으로써 30° 정도 위상이 늦게 한다. 비행검사관으로부터 보고된 경로 폭은 0.74°이다.
동축 아답터를 하단 안테나에 삽입하여 상대적으로 상단 안테나 전송선의 위상을 동일한 양으로 앞서게 한다. 이때 보고된 경로 폭은 0.92°이다.
조정해야 하는 SBO 신호의 위상을 계산하라.

풀이:

1
측정값을 Calculator 곡선에 표시
• W₁ = 0.74° → 곡선 왼쪽과 교점: θ₁ ≈ −21°
• W₂ = 0.92° → 곡선 오른쪽과 교점: θ₂ ≈ +42°
2
대칭성 점검 (|W₁ − W₂| 확인)
|0.74 − 0.92| = 0.18° ≫ 0.04°
→ 위상 어긋남 — 조정 필요
3
SBO 위상 오차 계산
방법 1 (교재 수식): 위상 오차 = (|θ₁| + |θ₂|) / 2 = (21 + 42) / 2 = 31.5°
의도적 지연 30°를 빼면 → 1.5° (이건 절대치 의미)

방법 2 (부호 포함): 위상 오차 = (θ₁ + θ₂) / 2 = (−21 + 42) / 2 = +10.5°
→ 상단 안테나의 SBO 위상이 정상보다 10.5° 앞섬
4
조정 방법
• 송신기 SBO 위상 조정기로 10.5° 늦춤
• 또는 SBO 측파대 케이블을 10.5°에 해당하는 약 1.75 cm 연장
• 조정 후 Dephasing Check 재실시

9-2. 연습문제

Q1. Quadrature Phasing 절차 이해
Quadrature Phasing 절차 중 "단계 2 — CSB Only 방사 시 DDM 측정"이 왜 기준값이 되는지 설명하시오.
답: CSB만 방사된 상태에서는 SBO 신호가 없으므로 SBO 위상이 어떤 값이든 DDM에 영향을 주지 않습니다.

→ 이 DDM 값은 "SBO가 제대로 작동하지 않는 상태"의 결과
→ 나중에 90° 케이블 삽입했을 때 (SBO 기여 0인 상태) DDM이 이 값과 같아야 함
→ 같다면 시스템의 자연 위상 오차가 0°라는 의미

수학적 근거: 90° 케이블 삽입 시 SBO 위상은 90°+θ_init가 됨. cos(90°+θ_init) ≈ −sin(θ_init). θ_init=0이면 SBO 기여 = 0 → CSB Only와 같은 DDM.
Q2. Dephasing Check 결과 해석
비행검사 결과: W₁ (상단 30° 지연) = 0.79°, W₂ (하단 30° 지연) = 0.83°. 위상 상태를 평가하고 조치를 결정하시오.
분석:
• |W₁ − W₂| = |0.79 − 0.83| = 0.04°
• 정확히 경계값에 위치

평가: 위상이 거의 정상이지만 경계 — 위상 오차가 약 ±2° 정도 있을 가능성

조치:
• 운용개시 검사인 경우: 0.04° 이내이므로 합격이지만, 가능하면 미세 조정 (이상적으로는 |W₁−W₂| < 0.02°)
• 정기검사 / 추가 보강: 패스폭 차이의 부호 확인 → W₂가 약간 크므로 SBO가 약간 앞서 있을 수 있음 → 약 2~3° 늦춤
• 운용 환경(CAT I vs II/III)에 따라 판단
Q3. Calculator 계산
측정값 W₁ = 0.78°, W₂ = 0.88°일 때 SBO 위상 오차를 계산하시오 (Calculator 사용).
풀이:
Calculator 곡선 식 W = 0.7 / cos(θ) → cos(θ) = 0.7 / W

W₁ = 0.78°에서:
cos(θ₁) = 0.7 / 0.78 = 0.897
θ₁ = ±arccos(0.897) = ±26.2°
왼쪽 (음의 영역): θ₁ ≈ −26°

W₂ = 0.88°에서:
cos(θ₂) = 0.7 / 0.88 = 0.795
θ₂ = ±arccos(0.795) = ±37.3°
오른쪽 (양의 영역): θ₂ ≈ +37°

SBO 위상 오차 = (θ₁ + θ₂) / 2 = (−26 + 37) / 2 = +5.5°
→ SBO가 5.5° 앞서 있음 → 송신기에서 5.5° 늦춤
Q4. 종합 시나리오 — 운용개시 검사 진행
신설 CEGP 시설에서 다음 결과:
• A단계 완료, B단계 Quadrature 완료 후 측정한 변조 균형: +3μA
• C단계 기준 패스폭: 0.71° (정상)
• D단계 W₁ = 0.85°, E단계 W₂ = 0.77°
현재 상태 평가 및 다음 조치는?
각 항목 평가:
• 변조 균형 +3μA: 운용개시 허용 ±5μA 이내 → 합격 ✓
• 기준 패스폭 0.71°: 0.70° ±0.07° 이내 → 합격 ✓
• Dephasing 차이: |0.85 − 0.77| = 0.08° → 0.04° 초과 → 위상 조정 필요 ✗

위상 오차 계산:
• W₁ = 0.85° (상단 30° 지연): cos(θ₁) = 0.7/0.85 = 0.824 → θ₁ ≈ ±34.5°
  상단 지연 = 음의 위상 영역 → θ₁ ≈ −34.5°
• W₂ = 0.77° (하단 30° 지연 = 양의 영역): cos(θ₂) = 0.7/0.77 = 0.909 → θ₂ ≈ +24.6°
• SBO 위상 오차 = (−34.5 + 24.6) / 2 = −5.0°
→ SBO가 5° 뒤짐 (Retarded)

조치 (F단계):
1. 송신기 SBO 위상 조정기로 5° 앞당김
2. C단계부터 재실시 (기준 패스폭 + Dephasing 재측정)
3. |W₁'−W₂'| ≤ 0.04° 도달까지 반복
4. 모든 항목 정상이면 G단계 (운용개시 승인)

비행시간 절약 팁: 5° 정도의 위상 오차는 비교적 작으므로 1회 보정으로 정상화 가능성 높음
Q5. 두 절차 차이 이해
"왜 Quadrature Phasing(지상)을 하고도 Dephasing Check(비행)를 또 해야 하는가?" 후배 기술자에게 설명해주시오.
설명:

① 측정 환경의 차이:
• Quadrature: 지상 미들마커 지점에서 PIR로 측정 — 근거리 효과 일부 포함, 지상 환경
• Dephasing: 비행기로 실제 활공로 비행 환경에서 측정 — 정확한 원거리 조건

② 측정 목적의 차이:
• Quadrature: 위상을 직접 조정 (송신기 SBO 위상기 설정값 만들기)
• Dephasing: 조정된 위상이 실제 비행 환경에서 정상인지 검증

③ 항공법 요구사항:
• 항공법 9번 항목 (Phasing): 변조 균형 ≤ 0μA → Quadrature로 달성
• 항공법 10번 항목 (Airborne Phase Verification): 실제 비행 환경에서 확인 → Dephasing으로 달성
두 항목은 서로 다른 절차로 검증해야 함

④ 시스템 관점:
• 지상 위상이 맞아도 안테나 마운트·케이블 연결·환경 등으로 원거리에서 미세 오차 가능
• 비행 검증을 통해 실제 항공기가 받는 신호가 정확한지 확인

한 줄 요약: "지상에서 맞추고, 비행으로 확인한다" — 한 쌍의 짝꿍 절차
