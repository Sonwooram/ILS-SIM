01
비행검사의 목적
항공기 실제 수신 신호 품질을 직접 확인하는 유일한 방법
⚡ 왜 비행검사가 필요한가?
지상 측정 장비로는 안테나 근처에서 신호를 측정한다. 그러나 실제 항공기가 수십km 떨어진 공중에서 수신하는 신호 품질은 지상에서 완전히 확인할 수 없다. 비행검사기가 실제 접근 경로를 비행하며 측정해야 한다.
지상 측정의 한계
  • 근접효과로 원거리 패턴 부정확
  • 항공기 수신 환경 미반영
  • Structure 등 원거리 특성 불가
비행검사로 확인
  • 실제 비행 중 DDM·코스폭 측정
  • 원거리에서의 구조(Structure) 분석
  • 전계강도 통달 범위 확인
운용자 역할
  • 비행검사 결과지 판독
  • 기본 조정 항목 식별
  • 감독하 조정 수행

Pass / Restricted / Unusable — 세 가지 판정

어느 항목(정렬·코스폭·Structure·Coverage)에서 이상인지 확인.

⚠️ 조정 시 주의사항
  • 모든 조정은 반드시 모니터 값 확인하면서 실시
  • 한 번에 한 가지 항목만 조정 — 동시 조정 금지

  • 조정 범위 초과 시 상위 정비자(전문 기술자) 요청
06
연습문제
버튼을 눌러 정답 확인
📝 연습문제
Q1. GP 비행검사 결과 패스폭이 1.6°(허용: 1.4°±0.21°)로 측정되었다. 판정과 조치 방향을 설명하라.
허용 범위: 1.4°±15% = 1.4°×0.85 ~ 1.4°×1.15 = 1.19° ~ 1.61°
측정값 1.6°는 허용 상한 1.61°와 매우 근접하여 Restricted 판정 가능.
조치: 패스폭이 넓다 → SBO(측파대) 신호 세기가 상대적으로 작다 → 분배망 조정으로 SBO 신호 증가 → 재측정으로 1.4° 근처로 조정 후 비행검사 재실시.
Q2. 비행검사 결과지에서 'Structure' 항목이 ±0.05 DDM으로 나타났다 (허용: ±0.031). 이것이 의미하는 바와 영향을 설명하라.
Structure는 코스/패스 선형 구간에서 DDM이 직선적으로 변하지 않고 굴곡지는 현상이다. ±0.05는 허용 ±0.031을 초과 → Restricted 또는 Unusable.
영향: 항공기 오토파일럿이 코스를 추적할 때 구불구불한 경로를 따라가게 되어 정밀 접근 품질 저하. 주원인은 다중 반사(Multi-path), 지형·구조물에 의한 반사파.
Q3. LOC 정기 비행검사에서 코스 정렬이 시단 기준 +15m (규정: ±10.5m) 편차로 측정됐다. 원인과 조치를 간략히 설명하라.
측정값 +15m > 허용 ±10.5m → Unusable 또는 즉시 조정 필요.
원인: LOC 좌우 SBO 신호 진폭 또는 위상 불균형 (한쪽이 더 강하면 코스가 치우침).
조치: 분배망에서 좌우 SBO 진폭 균형 조정. 조정 후 지상 측정으로 임시 확인 → 비행검사로 최종 검증. 조정 중 관제탑에 LOC 비정상 상태 통보.
