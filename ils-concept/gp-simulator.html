📐 Null Reference GP 원리
• 하단 안테나 (h): 변조된 반송파 신호(CSB) 방사. 높이 h = λ/(4·sinθ) — 활공각에 최대값
• 상단 안테나 (H = 2h): 측파대만(SBO) 방사. 활공각에 Null 형성
• 두 신호가 공간에서 합쳐져 활공각 위/아래로 대칭적 DDM 분포 형성
• CDI 동작: 활공각 위 → 니들 아래 (FLY DOWN) / 활공각 아래 → 니들 위 (FLY UP)

🛬 접근 단면도 해설
• GP 안테나 마스트 — 활주로 시단 측면 (250~600ft), Lower/Middle/Upper 안테나 위치
• ● ON PATH — DDM=0, 정상 활공각 (3°) 라인 위
• ▼ FLY DOWN 영역 — 90Hz 우세, 활공각 위쪽 (고도 낮춰야 함)
• ▲ FLY UP 영역 — 150Hz 우세, 활공각 아래쪽 (고도 높여야 함)
• ⚠ FALSE PATH (6°/9°/12°) — SBO Null이 정수배(2θ, 3θ, 4θ)에 형성 → 잘못된 활공각
• TCH (Threshold Crossing Height) — 활주로 시단 통과 고도, 일반 50ft
• 항공기 수신기는 코스 신호의 위상 반전을 인식해서 False Path 회피 (Flag Alarm)

🌐 GP 3D 패턴 해설
• 패턴 원점 = 지면: 이미지 안테나 효과로 안테나+반사파가 이미 합성된 결과 → lobe는 지면에서부터 펼쳐짐
• 수직면 lobing: 활공각(3°), 6°, 9°에 lobe/null 형성 → 얇은 부채꼴 모양이 정상
• LOWER (h): 활공각에 첫 lobe peak — CSB 최대점
• UPPER (2h/3h): 활공각에 null — SBO 형성의 핵심
• 수평면: 활주로 방향(+X)으로 광대역 방사 (좌우 ±60°)
• 합성 신호 토글 시 실제 CSB/SBO/CLR 3D 형상 표시
