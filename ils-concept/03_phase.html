01
왜 위상(Phase)을 알아야 하는가?
일반 AM 통신은 위상이 필요 없다 — 그런데 ILS는 왜 다른가?
⚡ 핵심 질문 먼저
라디오 방송은 위상을 신경 쓰지 않는다. 그런데 ILS는 위상이 조금만 틀려도 항공기가 잘못된 방향으로 유도된다. 왜 그럴까?
📻 일반 AM 통신 (단일 안테나)

단 하나의 안테나에서 반송파 + 측파대 신호가 함께 방사된다.

  • 두 신호가 항상 같은 위치에서 출발 → 위상관계 자동 유지
  • 수신기는 그냥 "얼마나 강한가"만 보면 됨
  • 위상은 신경 쓸 필요 없음
✈️ ILS / 배열안테나 (분리 방사)

CSB(반송파+측파대)와 SBO(측파대만)가 서로 다른 안테나에서 방사된다.

  • 두 신호가 다른 경로로 공간을 날아감
  • 수신기 위치에 따라 도달 거리가 달라짐 → 위상차 발생
  • 위상차가 변조량(DDM)을 결정 → 항공기 위치 정보!
💡 한 문장 결론
위상 = 공간에서 두 신호가 "얼마나 같은 시점에 도착하는가"를 나타내는 지표이며, 이것이 ILS에서는 항공기의 좌우·상하 위치 정보가 된다.

위상이 만들어내는 것 — 공간에서의 합성

두 안테나에서 동일한 신호를 방사하면, 공간의 각 지점에서 두 신호가 합쳐진다. 그 합쳐진 결과가 위상에 따라 극적으로 달라진다:

위상에 따른 합성 결과
02
위상의 기초 개념
사인파, 위상각, 위상차 — 수식 전에 직관부터

2-1. 사인파와 위상각

모든 RF 신호는 사인파다. 사인파는 진폭(A), 주파수(f), 위상(φ) 세 가지로 완전히 표현된다.

사인파 기본 수식
v(t) = A · sin(ωt + φ)
A = 진폭 (최대 전압/전류)
ω = 2πf = 각속도 (rad/s)
φ = 위상각 (초기 위상, 단위: °또는 rad) ← 이것이 핵심!
t = 시간
위상천이(Phase Shift)된 사인파
📌 위상 지연 vs 위상 진행

φ가 양수(+) → 위상이 앞선다 (Phase Lead): 기준보다 먼저 최대값 도달

φ가 음수(-) → 위상이 늦는다 (Phase Lag): 기준보다 나중에 최대값 도달

거리가 길수록 위상이 늦는다 — 이것이 ILS에서 공간 위상차의 핵심 원인!

2-2. 거리와 위상 지연의 관계

전자기파는 빛의 속도(약 3×10⁸ m/s)로 전파된다. 거리가 멀수록 도달하는 시간이 늦어지고, 이것이 위상 지연으로 나타난다.

거리 → 전기적 위상각 변환
β = (거리 / λ) × 360°
β = 위상 지연각 (°)
λ = 파장 (m) = 빛의 속도 / 주파수
예: ILS LOC 110MHz → λ = 3×10⁸ / 110×10⁶ ≈ 2.73m
안테나에서 2.73m 떨어진 지점 → 위상 360° 지연 (= 한 파장)

2-3. 두 안테나에서 방사될 때의 위상차

두 개의 안테나가 간격 2a 만큼 떨어져 있을 때, 공간의 어느 지점(각도 θ)에서의 두 신호의 상대 위상차는:

안테나 배열 위상차 공식
경로차 = a·sin(θ) → 위상차 = a·sin(θ) [전기각]
a = 안테나 간 반거리 (전기각 단위)
θ = 관측 방향각 (0° = 정면)
θ = 0° (정면): sin(0°)=0 → 경로차 0 → 위상차 없음 → 최대 합성
θ = 90° (측면): sin(90°)=1 → 경로차 최대 → 위상차 최대
두 안테나의 경로차에 의한 위상차 발생 원리
03
페이서(Phasor) — 위상을 그림으로 보는 법
복잡한 계산을 벡터 화살표 하나로 해결하는 강력한 도구
🎯 페이서란?
사인파를 회전하는 화살표(벡터)로 표현한 것. 화살표의 길이 = 진폭, 각도 = 위상. 여러 신호를 합칠 때 벡터 덧셈만 하면 끝!
페이서 표현법 — 사인파 → 화살표

3-1. 페이서로 신호 합성하기

두 신호를 합칠 때, 각각의 페이서를 벡터 덧셈하면 합성 신호의 진폭과 위상을 한눈에 볼 수 있다.

위상각에 따른 페이서 합성 — 3가지 경우

3-2. ILS에서 극좌표 표현

교재에서 사용하는 표현 방식: I∠φ = 진폭 I, 위상각 φ인 신호

표현의미예시
1∠0° 진폭=1, 위상=0° (기준 신호) CSB 반송파
1∠90° 진폭=1, 위상=90° 앞섬 A₁에서 가까운 쪽 신호
1∠-90° 진폭=1, 위상=90° 늦음 A₂에서 먼 쪽 신호
1∠180° 진폭=1, 위상이 완전 반전 SBO: Out-of-phase 공급
04
배열안테나와 위상 — 방사패턴 형성
In-phase / Out-of-phase 공급이 만드는 완전히 다른 패턴

배열안테나는 여러 안테나에 위상과 진폭을 조절하여 신호를 공급함으로써 원하는 방향에 강한 신호를, 원하지 않는 방향에 Null을 만든다. 이것이 ILS 코스·패스를 형성하는 핵심이다.

4-1. 동위상 쌍 (SIP: Specific In-phase Pair)

SIP 특성
두 안테나에 같은 위상(0°)의 전류 공급 → 정면(0°) 방향 최대, 측면 Null 가능
SIP 합성 전계강도
E_SIP = 2I · cos(a·sinθ)
a = 안테나간 반거리 (전기각)
θ = 관측 방향각
최대: θ=0° (정면) → cos(0)=1 → E=2I
Null: a·sinθ = ±90° 일 때 → cos(90°)=0

4-2. 역위상 쌍 (SOP: Specific Out-of-phase Pair)

SOP 특성
두 안테나에 반대 위상(180°)의 전류 공급 → 정면(0°)에서 Null, 측면에서 최대
SOP 합성 전계강도
E_SOP = 2I · sin(a·sinθ)
Null: θ=0° (정면) → sin(0)=0 → E=0 ← 항상 정면 Null!
최대: a·sinθ = ±90° 일 때 → sin(90°)=1 → E=2I
SIP vs SOP 방사패턴 비교
🔑 ILS 적용 핵심

CSB 안테나 → SIP
정면 방향으로 강한 신호 → 어디서나 수신 가능한 기준 신호

SBO 안테나 → SOP
정면(중심선)에서 Null → 좌우로 벗어나면 강도 증가 → 위치 정보!

4-3. 방사패턴의 위상 구분

방사패턴의 로브는 위상도 가지고 있다. 이것이 CSB+SBO 합성 시 위치별로 다른 변조도를 만든다.

패턴 종류특징예시
단상(Uni-phase) 모든 로브가 같은 위상 (0°) SIP 패턴 (모든 방향 동위상)
양상(Bi-phase) 로브마다 위상이 다름 (0°, 180°) SOP 패턴 — 좌측 로브와 우측 로브가 역위상
가변위상(Variable-phase) 로브 내에서 위상이 변함 다중 배열의 일부 방향
⚡ SBO의 비밀 — 왜 좌우가 "반대 위상"인가?
SOP 패턴에서 정면(0°)을 기준으로 왼쪽 로브와 오른쪽 로브는 180° 역위상 관계다.
이것이 CSB와 합쳐졌을 때, 왼쪽에서는 90Hz가 우세하고 오른쪽에서는 150Hz가 우세하게 만드는 핵심 원리!
05
공간변조(Space Modulation)와 위상
CSB + SBO가 공간에서 만나 위치 정보를 만드는 과정
💡 공간변조란?
반송파 신호(CSB)와 측파대 신호(SBO)를 별도의 안테나에서 분리 방사하고, 수신기 위치에서 두 신호가 공간에서 합성되어 AM 신호를 형성하는 기술. 이때 수신기 위치(각도 θ)에 따라 위상관계가 달라지면서 변조도가 달라진다.

5-1. 유효 공간변조지수 (Sf)

공간에서 CSB와 SBO가 합성될 때, 실제로 수신기에서 인식되는 변조도는 두 신호간의 위상차 φ에 영향을 받는다.

유효 공간변조지수
S_f = (E_ssm · cosφ) / E_cm
E_ssm = 공간에서 수신된 분리 방사 측파대 신호의 진폭
E_cm = 공간에서 수신된 반송파 신호의 진폭
φ = 두 신호간 위상각
cosφ가 핵심! — 위상이 맞으면 최대, 90°면 0, 180°면 음수
위상각 φcosφS_f 결과의미
φ = 0° 1.0 S_f = 최대 완전 동위상 — 정상 변조
φ = 45° 0.707 S_f = 70.7% 변조도 감소
φ = 90° 0 S_f = 0 직각위상 — 변조 없음! 코스폭 무한대
φ = 180° -1.0 S_f = 최대(역) 완전 역위상 — 변조 반전 (송신기변조와 감산)

5-2. 총 변조지수 = 송신기 변조 + 공간변조

총 변조지수 계산
M_total = m + S_f    (동위상일 때)
M_total = m − S_f    (역위상일 때)
m = 송신기 변조지수 (단일 안테나에서의 변조도)
S_f = 유효 공간변조지수
동위상(+): 두 변조가 합산 → 총 변조 증가
역위상(-): 두 변조가 상쇄 → 총 변조 감소

5-3. 공간에서 위치별 90Hz/150Hz 우세도

ILS 로컬라이저 공간변조 — 위치별 DDM 형성
🔬 SBO가 좌우 반대 위상인 이유

SBO는 SOP(역위상) 안테나 쌍에서 방사되므로, 정면(중심선)을 기준으로 좌측 로브와 우측 로브가 180° 역위상이다.

따라서:

  • 왼쪽: SBO 로브가 CSB와 동위상 → 특정 주파수(90Hz) 변조 합산 증가
  • 오른쪽: SBO 로브가 CSB와 역위상 → 90Hz 변조 감소, 150Hz 변조 합산 증가
06
위상 오차가 ILS에 미치는 영향
위상이 조금만 틀려도 — 코스폭, DDM, 항공기 유도 오차
⚠️ 위상 오차의 심각성
CSB와 SBO 사이의 위상이 90°가 되면 모든 위치에서 DDM=0이 된다. 즉 항공기가 어디 있어도 "중심선"으로 인식 → 유도 불가!

6-1. SBO 위상 오차가 DDM에 미치는 영향

SBO의 위상이 정상값에서 벗어나면, DDM은 위상차 φ에 cosφ를 곱한 만큼 감소한다:

위상 오차에 의한 DDM 변화
DDM_실제 = DDM_이상 × cos(φ_error)
φ_error = SBO와 CSB 사이의 위상 오차
위상 오차가 커질수록 DDM이 줄어들고 → 코스폭이 넓어지고 → 결국 코스 불명확
위상 오차에 따른 DDM 감소 (cosφ 함수)

6-2. 위상 오차의 실제 원인들

📡 전송선 길이 차이

CSB와 SBO 안테나까지의 케이블 길이가 다르면 도달 위상이 달라짐

🏗️ 근접 물체 반사

건물, 항공기, 지형물에서 반사된 신호가 합쳐져 위상 변화 유발

🌍 수신 위치 경로차

다른 안테나에서 방사된 두 신호가 수신기에 다른 경로로 도달 → 자연적 위상차

6-3. 90° 위상차(Quadrature Phase)의 특별한 경우

CSB와 SBO가 90° 위상차로 합성될 때, 수신기에서 변조가 0이 된다. 그러나 이때 2차 고조파 성분은 최대가 된다는 점에 주의.

📐 직각위상(Quadrature) 요약
  • cos(90°) = 0 → 기본 변조 주파수 성분 = 0
  • 2차 고조파(180Hz, 300Hz) 성분은 이 조건에서 최대
  • 수신기 필터가 기본 주파수만 통과 → 결국 오디오 출력 없음
  • DDM = 0 → 항공기 계기 중심으로 → 모든 위치에서 "코스 위"로 오인
07
실무 적용 — 점검과 조정
위상 지식을 현장에서 어떻게 사용하는가

7-1. RF 위상 점검의 중요성

ILS 시설에서 위상 점검은 코스/패스 신호의 품질을 보장하는 핵심 작업이다.

🔧 점검 포인트
  • CSB-SBO 간 RF 위상 관계 확인
  • 안테나 분배망(Distribution Unit) 위상 조정
  • 전송선 길이에 의한 위상 보정
  • 모니터 시스템의 위상 감시 기능
📊 이상 징후
  • 코스폭(Course Width)이 규정보다 넓어짐
  • DDM 값이 이론값보다 낮게 측정
  • 중심선에서 벗어난 위치의 DDM 변화율 감소
  • 비행검사에서 코스 감도 저하

7-2. 위상과 DDM의 관계 — 실무 계산

DDM 계산 (기본)
DDM = (m₉₀ − m₁₅₀) / 100
m₉₀ = 90Hz 변조도 (%)
m₁₅₀ = 150Hz 변조도 (%)
중심선(DDM=0): m₉₀ = m₁₅₀
위상이 틀리면: m₉₀과 m₁₅₀ 모두 줄어들어 DDM 감도 저하

7-3. 근접효과(Proximity Effect)와 위상

측정 장비나 수신기를 안테나에 너무 가까이 놓으면(P < 100a), 두 안테나까지의 경로차 계산이 원거리 근사식과 달라진다. 이로 인해 위상 오차가 생겨 DDM 측정값이 부정확해진다.

📏 원거리 근사 조건

P ≥ 100a 일 때만 asinθ 근사식이 유효 (P = 안테나 배열 중심에서 수신점까지 거리)

이 조건을 만족할 때 측정해야 정확한 DDM/위상 데이터를 얻을 수 있다.

7-4. 종합 흐름도 — 위상에서 항공기 유도까지

위상 → 방사패턴 → 공간변조 → DDM → 항공기 유도 흐름
1
송신기에서 CSB·SBO 신호 생성

90Hz·150Hz 변조된 CSB와 반송파 억압된 SBO 신호를 각각 생성. 이때 위상 관계 설정.

2
분배망을 통해 각 안테나에 공급 (SIP/SOP)

CSB → SIP(동위상) 안테나 쌍, SBO → SOP(역위상) 안테나 쌍. 위상·진폭 정밀 배분.

3
각 안테나가 고유 방사패턴으로 신호 방사

CSB: 전방위 균등(Uni-phase), SBO: 중심선 Null, 좌우 역위상(Bi-phase)

4
공간에서 위치별 위상차로 합성 → 변조도 차이 발생

수신 위치(θ)에 따라 두 신호의 위상관계 변화 → 유효 공간변조지수 S_f 결정

5
항공기 수신기가 DDM 측정

DDM = 90Hz 변조도 - 150Hz 변조도. 이 값으로 계기 지침 위치 결정.

6
조종사/오토파일럿이 중심선으로 유도

DDM=0 → 중심선, DDM>0 → 왼쪽 벗어남, DDM<0 → 오른쪽 벗어남

📝 연습문제 — 개념 확인
Q1. 일반 AM 방송에서는 위상을 신경 쓰지 않아도 되는 이유는 무엇인가?
반송파와 측파대가 동일한 하나의 안테나에서 방사되기 때문에, 두 신호는 항상 같은 위치에서 출발하고 위상관계가 자동으로 유지된다. ILS처럼 별도 안테나에서 분리 방사하지 않으므로 공간에서의 위상 변화가 문제가 되지 않는다.
Q2. SOP(역위상) 안테나 쌍의 방사패턴에서 0°(정면) 방향에 항상 Null이 생기는 이유를 위상의 관점에서 설명하라.
0° 방향(정면)에서는 두 안테나로부터의 전파 경로가 동일하여 경로차가 0이다. 따라서 안테나에 공급된 전류의 위상차(180°)가 그대로 수신점에 나타난다. 180° 역위상인 두 신호가 합성되면 서로 완전 상쇄되어 합성 전압 = 0, 즉 Null이 된다. (E_SOP = 2I·sin(a·sin0°) = 2I·sin(0) = 0)
Q3. CSB 신호 진폭 100V, SBO 신호 진폭 20V일 때, 두 신호의 위상차가 각각 (a) 0°, (b) 60°, (c) 90°일 때의 유효 공간변조지수 S_f를 구하라.
공식: S_f = (E_ssm · cosφ) / E_cm

(a) φ=0°: S_f = 20·cos0° / 100 = 20×1 / 100 = 0.2 (20%)
(b) φ=60°: S_f = 20·cos60° / 100 = 20×0.5 / 100 = 0.1 (10%)
(c) φ=90°: S_f = 20·cos90° / 100 = 20×0 / 100 = 0 (0%)

→ 90° 위상차에서는 공간변조가 완전히 사라져 DDM=0이 되며, 코스폭이 무한대가 된다.
Q4. LOC 코스 왼쪽에서 90Hz가 우세하고, 오른쪽에서 150Hz가 우세한 이유를 SBO의 위상 특성으로 설명하라.
SBO 안테나 쌍은 SOP(역위상) 방식으로 구동되므로 방사패턴이 Bi-phase다. 즉, 중심선 기준 왼쪽 로브와 오른쪽 로브가 180° 역위상 관계다.

• 왼쪽: SBO 로브가 CSB의 90Hz 측파대와 동위상 → 90Hz 변조 합산 증가
• 오른쪽: SBO 로브가 CSB의 90Hz 측파대와 역위상 → 90Hz 변조 감소, 150Hz 변조 합산 증가

이 차이가 항공기 수신기에서 DDM으로 나타나 좌우 위치 판단에 사용된다.
Q5. 두 안테나가 전기적 간격 a=90°인 SIP로 구성되어 있다. θ=30° 방향에서의 합성 전계강도는 기준값(정면 최대)의 몇 %인가?
공식: E_SIP = 2I·cos(a·sinθ)
정면 최대값: 2I·cos(90°·sin0°) = 2I·cos(0) = 2I (= 100%)
θ=30°: E = 2I·cos(90°·sin30°) = 2I·cos(90°×0.5) = 2I·cos(45°)
= 2I × 0.707 = 1.414I → 약 70.7%

θ=30° 방향에서는 정면 대비 약 70.7%의 신호 세기를 보인다.
