01
페이서(Phasor)란?
회전하는 화살표 — 사인파의 진폭과 위상을 동시에 표현
⚡ 왜 페이서가 필요한가?
ILS에서는 여러 안테나의 신호가 공간에서 합쳐진다. 수식으로 계산하면 복잡한 삼각함수 연산이 필요하지만, 페이서(벡터 화살표)로 표현하면 그림 위의 벡터 덧셈으로 순식간에 해결된다.

1-1. 페이서의 정의

사인파 v = V_p · sin(ωt + φ)를 크기(길이) = V_p, 각도 = φ인 화살표로 표현한 것.

페이서가 반시계 방향으로 360° 회전하는 것이 사인파 한 주기(360°)에 해당한다.

사인파 → 페이서 변환 원리
📌 페이서의 3가지 규칙
  • 길이 = 신호의 진폭(최대값 V_p)
  • 각도 = 위상각 φ (기준선 X축에서 반시계 방향)
  • 반시계 회전 = 위상 증가 방향 (관례)
02
극좌표 표현 — I∠φ
ILS 교재 전체에서 사용하는 표현 방식

ILS 교재에서는 안테나 전류를 I∠φ 형태로 표현한다.

극좌표 표현
Ȧ = A∠φ = A(cosφ + j·sinφ)
A = 진폭 (크기)
φ = 위상각 (°)
j = 허수 단위 (√-1)
표현의미ILS 사용 예
1∠0°기준 신호 (위상 0°)반송파 기준
1∠90°90° 앞서는 신호가까운 안테나 신호
1∠-90°90° 늦는 신호먼 안테나 신호
1∠180°완전 역위상SBO: Out-of-phase 공급
03
벡터 합성 — 두 신호 더하기
tip-to-tail 법칙으로 합성 신호의 진폭과 위상을 한눈에

두 신호를 합칠 때 각각의 페이서를 끝점에서 꼬리로(tip-to-tail) 이어 붙이면 합성 페이서를 구할 수 있다.

위상각에 따른 벡터 합성 결과 — 4가지 경우
두 페이서의 합성 크기 (코사인 법칙)
|E_t| = √(E₁² + E₂² + 2·E₁·E₂·cosθ)
θ = 두 페이서 사이의 각도
θ=0°: |E_t| = E₁+E₂ (최대)
θ=90°: |E_t| = √(E₁²+E₂²)
θ=180°: |E_t| = |E₁-E₂| (최소, 동일 크기면 0)
04
안테나 배열에서의 위상 비교
두 안테나의 신호가 관측점에서 만날 때의 위상 계산

두 안테나 A₁·A₂에서 관측점 P까지의 경로가 다르면, 두 신호의 위상도 달라진다.

안테나 배열 각 신호의 위상 (교재 방정식 2-5·2-6)
β₁ = φ₁ + a·sinθ    (A₁에서 P까지)
β₂ = φ₂ − a·sinθ    (A₂에서 P까지)
φ₁, φ₂ = 각 안테나 전류의 위상
a = 안테나간 반거리 (전기각 단위)
θ = 관측 방향각
θ=0°이면 a·sinθ=0 → 경로차 없음 → 위상차는 전류 위상차만 반영
💡 ILS 안테나 배열 계산 핵심

각 안테나의 전류 위상(φ₁, φ₂)과 관측 방향(θ)을 알면, 합성 전계강도를 페이서 덧셈으로 계산할 수 있다. 이것이 방사패턴 계산의 기본 과정이다.

SIP(동위상): φ₁=φ₂=0° → E_t = 2I·cos(a·sinθ)
SOP(역위상): φ₁=0°, φ₂=180° → E_t = 2I·sin(a·sinθ)

05
AM파의 페이서 표현
반송파·상측파대·하측파대 — 3개 페이서의 회전

진폭변조(AM) 신호는 세 개의 페이서로 구성된다. 반송파 페이서를 기준(정지)으로 보면, 상측파대는 반시계, 하측파대는 시계 방향으로 회전한다.

반송파 (Carrier)

각속도: ω_c
크기: E_cm
위상: 0° (기준)

상측파대 (USB)

각속도: ω_c + ω_a
크기: m·E_cm/2
반송파 기준 반시계 회전

하측파대 (LSB)

각속도: ω_c − ω_a
크기: m·E_cm/2
반송파 기준 시계 회전

📌 상·하측파대는 항상 대칭
상측파대와 하측파대 페이서는 반송파 페이서를 기준으로 항상 같은 양의 각도만큼 반대 방향으로 회전한다. 두 측파대 페이서의 합(총 측파대)은 항상 반송파와 동위상 또는 역위상이다 — 이것이 정상적인 AM 신호의 조건이다.

07
연습문제
버튼을 눌러 정답을 확인하세요
📝 연습문제
Q1. 페이서 1∠30°와 1∠-30°를 합성하면 합성 페이서의 크기는? (코사인 법칙 사용)
두 페이서 사이의 각도 θ = 60°
|E_t| = √(1² + 1² + 2·1·1·cos60°) = √(1+1+2×0.5) = √3 ≈ 1.732
방향: 두 페이서의 중간이므로 0° (정면). 즉 E_t = 1.732∠0°
Q2. SOP(역위상) 안테나 쌍에서 θ=0° (정면) 방향의 합성 전계강도가 0인 이유를 페이서로 설명하라.
θ=0°이면 경로차 = a·sin0° = 0. 따라서 두 안테나 신호가 P점에 도달할 때 추가 위상차는 없다. 안테나 전류가 이미 180° 역위상이므로 두 페이서는 정반대 방향 → 벡터합 = 0. 페이서로: E₁ = 1∠0°, E₂ = 1∠180° → E_t = 1∠0° + 1∠180° = 0
Q3. AM 신호에서 상측파대와 하측파대 페이서의 합(총 측파대)이 항상 반송파와 동위상 또는 역위상인 이유는?
상측파대는 반송파 기준 반시계, 하측파대는 시계 방향으로 같은 각속도(ω_a)로 회전한다. 두 페이서가 항상 반송파를 기준으로 대칭이므로, 벡터합은 반드시 반송파 방향(0° 또는 180°)에 정렬된다. 이것이 AM 신호 포락선이 순수하게 변화하는 조건이다.
