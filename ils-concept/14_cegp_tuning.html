01
CEGP 신호 구성 복습
조정하기 전에 어떤 신호들이 있는지 명확히 알아야 합니다
⚡ 이 강의의 목표
비행검사 결과지를 받았을 때 "어느 항목이 어떻게 안 나왔는가" → "어떤 신호를 어떻게 조정해야 하는가"를 직관적으로 판단할 수 있도록 만드는 것이 목표입니다.
각 조정 항목의 왜(원리) + 어떻게(절차) + 얼마나(허용범위)를 한 번에 정리합니다.

1-1. CEGP의 3대 신호

CEGP(Capture Effect Glide Path)는 NRGP와 다르게 3가지 신호로 구성됩니다. 각 신호가 어떤 안테나에서 방사되는지, 어떤 역할을 하는지 명확히 구분해야 조정이 가능합니다.

CSB (Course)
중단(M) + 하단(L)
코스 안테나 페어
반송파 + 두 측파대(90/150Hz)가 합성된 신호. 활공각의 기준이 되는 신호. 활공각 위에서 90Hz·150Hz 균형, 위/아래로 비대칭 발생.
SBO (Course)
상단(U) + 중단(M) + 하단(L)
모든 안테나 (진폭비 −0.5:1:−0.5)
측파대 단독 신호. 3개 안테나 모두로 분배되며 중단을 기준으로 상단·하단은 역위상에 절반 진폭. 패스폭(DDM 기울기)과 활공각 형성에 동시 기여.
CLR (Clearance)
상단(U) + 하단(L)
클리어런스 안테나 페어
반송파에서 −8kHz 떨어진 별도 주파수. 150Hz만 80% 변조. 활공각 아래 영역에서 Fly-Up 보장. 강도는 코스 신호의 15% 이하.

CEGP 안테나 배열과 신호 분배

1-3. 신호 합성 — 어디서 무엇이 만들어지나

CEGP의 핵심은 각 신호가 공간에서 합쳐져서 활공경사를 형성한다는 점입니다. 각 신호의 역할을 표로 정리하면:

신호 주파수·변조 방사 안테나 담당 역할
CSB f₀ + 4kHz
90Hz + 150Hz (40%)		 중단(M) + 하단(L)
동위상, h:2h		 활공각 위에서 균형 → 활공각 결정
SBO f₀ + 4kHz
90Hz · 150Hz (역위상)		 상단(U) + 중단(M) + 하단(L)
진폭비 −0.5 : 1 : −0.5		 활공각에서 Null + 패스폭 결정
(중단이 주, 상·하단이 보조)
CLR f₀ − 4kHz
150Hz만 (80%)		 상단(U) + 하단(L)
동위상, h:3h		 활공각 아래쪽 Fly-Up
클리어런스 보장
📌 NRGP와의 결정적 차이
NRGP: 안테나 2개(L, U), 신호 2개(CSB, SBO) → 평탄지에서 단순 동작
CEGP: 안테나 3개(L, M, U), 신호 3개(CSB, SBO, CLR) → 평탄지 부족 환경(불규칙 지형)에서 사용
그래서 CEGP는 조정 항목과 영향 관계가 NRGP보다 복잡합니다. 같은 비행검사 항목 이상도 CEGP에서는 더 다양한 원인이 가능해요.
⚠️ CEGP SBO 신호 분배 — 흔한 오해
SBO 신호는 NRGP처럼 안테나 2개에 공급되는 것이 아닙니다.
교재 13장 명시: "활공각의 형성은 중단 안테나가 형성하는 Null 뿐만 아니라, 상단과 하단 안테나에 공급되는 분리 방사되는 측파대 신호의 위상과 진폭에도 영향을 받는다."

즉 CEGP에서 SBO는:
• 상단(U)에 −0.5 진폭 (역위상)
• 중단(M)에 +1.0 진폭 (기준)
• 하단(L)에 −0.5 진폭 (역위상)
이렇게 3개 모두에 공급됩니다. 활공각 위치는 단순히 중단 Null만이 아니라 3개 안테나 SBO의 공간 합성 결과로 형성됩니다.
→ 그래서 SBO 진폭/위상 조정은 패스폭뿐 아니라 활공각·대칭성에도 미세 영향을 줍니다.

2-2. 핵심 원리 — 왜 이렇게 매핑되는가

이 매트릭스가 왜 이렇게 되는지 원리적으로 이해해야 응용이 가능합니다.

📐활공각 = 안테나 높이가 주, SBO도 미세 영향

Null 위치 = h = λ/(4·sinθ)
→ 안테나 높이가 1차 결정 요소.
CEGP는 SBO가 3개 안테나에 분배되므로 SBO 위상·진폭, 변조 균형도 활공각 미세 위치에 영향. 큰 조정은 높이로, 미세 조정은 SBO로.

📏패스폭 = SBO 진폭의 함수

DDM 기울기는 SBO/CSB 비율에 비례. SBO 진폭을 키우면 DDM 기울기가 가파르게 됨 → 패스폭 좁아짐.
반대로 SBO 진폭을 줄이면 패스폭 넓어짐.

🔄위상 = SBO 위상의 함수

시스템 페이징은 CSB와 SBO의 위상 관계를 점검. SBO 위상이 어긋나면 DDM이 비대칭이 되거나 활공각 자체가 어긋남.

🌊클리어런스 = CLR 진폭의 함수

활공각 아래쪽 Fly-Up 신호 강도는 CLR 진폭으로 결정. 너무 약하면 클리어런스 부족, 너무 강하면 코스 구간(Structure)에 간섭.

03
활공각 (Path Angle) 조정 ★
"활공각이 어긋났다" → 안테나 높이를 조정한다

3-1. 원리 — 왜 안테나 높이가 활공각을 결정하는가

활공각 형성의 핵심은 중단 안테나(Middle)가 만드는 첫 번째 Null의 위치입니다.

FORMULA — 활공각과 안테나 높이
sin θ = λ / (4 × h_L)
여기서 θ는 활공각, λ는 파장(GP 330MHz → 약 0.91m), h_L은 하단 안테나 높이.
→ 안테나 높이가 높아지면 활공각이 낮아지고, 낮아지면 활공각이 높아진다.
CEGP: 하단:중단:상단 = h:2h:3h (1:2:3 비율 유지)

3-2. 실무 환산 — 높이 1cm 차이 = 활공각 얼마?

활공각 3.0° 시설(λ=0.91m)에서 표준 하단 안테나 높이는 약 4.34m입니다. 안테나 높이 변화에 따른 활공각 변화:

📌 활공각 조정 원칙
• CEGP는 3개 안테나의 높이비 (1:2:3)를 유지하면서 전체적으로 마스트의 베이스 위치(또는 안테나 세트의 절대 높이)를 조정합니다.
• 한 안테나만 따로 조정하면 안 됩니다 (높이비 깨짐 → 다른 항목 다 어긋남).
• 실무적으로는 안테나 마운팅 브라켓의 슬롯으로 mm 단위 조정 가능.

3-3. 조정 절차

1
현재 활공각 측정 → 목표 활공각 차이 계산
예: 측정 3.12°, 목표 3.00° → 차이 +0.12° (활공각이 0.12° 높음)
2
필요한 안테나 높이 변화량 계산
Δh ≈ h × (Δθ / θ)
예: 4.34m × (0.12 / 3.00) = +0.174m = +17.4cm (안테나를 17.4cm 올려야 함)
3
3개 안테나 동시 조정 (1:2:3 비율 유지)
• Lower(h): +17.4cm 올림
• Middle(2h): +34.8cm 올림
• Upper(3h): +52.2cm 올림
→ 각각의 절대 높이가 정확히 1:2:3 유지되도록
4
지상 측정 후 비행검사 재실시
• 지상에서 NF Monitor 확인 (활공각 정상 범위)
• 비행검사로 활공각, 패스폭, 위상 모두 재측정
⚠️ 활공각 조정 시 부수 영향
안테나 높이를 바꾸면 활공각 외에 패스폭과 위상에도 영향이 갑니다. 따라서:
1. 활공각 먼저 조정 → 2. 패스폭 재측정·조정 → 3. 위상 재점검·조정 → 4. 클리어런스 재확인
이 순서를 지켜야 합니다.

3-4. 실제 사례 — Minneapolis-St. Paul 공항

🛬 ICASC 사례 연구
상황: Minneapolis-St. Paul Runway 12L가 CAT III 업그레이드. 활주로 시단으로부터의 안테나 거리 제약(택시웨이 근접)으로 기준점 높이(RDH) 63ft로 측정 → 허용 한계 초과.

해결: 활공각을 3.0° → 2.8°로 낮춤. 이를 위해 안테나 마스트 전체 높이를 약간 높임. 결과적으로 RDH가 허용 범위로 회귀하면서 CAT III 운용 가능.

교훈: 활공각 조정은 단순히 활공각 자체뿐 아니라 RDH(시단 통과 높이)도 고려해야 합니다. 활공각과 안테나 위치(거리)가 함께 RDH를 결정.

04
패스폭 (Path Width) 조정 ★
"패스폭이 안 맞는다" → SBO 신호 진폭을 조정한다

4-1. 원리 — 왜 SBO 진폭이 패스폭을 결정하는가

패스폭은 DDM의 기울기로 결정됩니다. DDM = ±0.175 지점이 바로 패스 경계인데, 이 지점이 활공각에서 얼마나 떨어져 있는지가 패스폭이에요.

FORMULA — 패스폭과 SBO/CSB 비율
DDM ∝ (SBO / CSB) × sin(elevation - θ)
활공각 근처에서 DDM은 (SBO/CSB) 비율에 비례해서 기울기가 결정됨.
→ SBO 진폭 ↑ = DDM 기울기 가파름 = 패스폭 좁아짐
→ SBO 진폭 ↓ = DDM 기울기 완만 = 패스폭 넓어짐
SBO 진폭 변화에 따른 DDM 기울기와 패스폭

4-2. SBO 진폭 조정 방법

SBO 진폭은 송신기 내부의 SBO 출력 레벨 조정기(Attenuator)로 조정합니다. 보통 dB 단위로 가변되며, 일부 시스템은 step attenuator (1dB·2dB·4dB·8dB 조합)로 구성됩니다.

1
측정된 패스폭과 목표 패스폭 비교
예: 측정 1.8°, 목표 1.4° → 패스폭이 0.4° 넓음 → SBO를 키워야 함
2
필요한 SBO 변화량 계산
dB 변화 = 20 × log₁₀(목표 폭 / 현재 폭)
예: 20 × log₁₀(1.4/1.8) = 20 × log₁₀(0.778) = −2.17 dB
→ SBO를 2.17 dB 강화 (감쇠기를 2dB 감소)
3
송신기 SBO 감쇠기 조정
• 송신기 캐비닛 내부 SBO Output Attenuator 조정 (보통 step type)
• 1dB 또는 2dB 단위로 조정
• 미세 조정이 필요하면 SBO 분배망의 위상 조정기도 함께 고려
4
재측정 및 미세 조정
• 비행검사 또는 PIR로 새 패스폭 측정
• 목표값에 도달할 때까지 반복

4-3. 안테나 높이 변경 시 패스폭에도 영향

📌 부수 영향
안테나 높이를 활공각 조정용으로 바꾸면 패스폭도 같이 변합니다. 안테나가 높아지면 패스폭이 좁아지고, 낮아지면 넓어집니다 (sin θ 의존성).
그래서 활공각 조정 → 패스폭 재측정 → 필요시 SBO 진폭으로 보정하는 순서로 진행합니다.
05
위상 (Phasing) 조정 — Quadrature Method ★
"DDM이 비대칭이다 / 위상이 어긋난다" → 측파대 위상 조정

5-1. 원리 — 시스템 페이징이란?

GP 시설의 위상이 정상이라는 의미는:

  • 측파대 안테나의 90Hz 성분이 반송파의 90Hz 성분과 정확히 동위상
  • 측파대 안테나의 150Hz 성분이 반송파의 150Hz 성분과 정확히 역위상 (180°)

이 관계가 맞지 않으면 공간에서 합성된 DDM이 비대칭이 되거나 활공각 위치가 어긋납니다.

🔄 Quadrature Phasing이란?
"Quadrature"는 90° 위상차를 의미. 위상 점검을 위해서 일부러 SBO에 90° 케이블을 끼워 인위적으로 90° 위상차 상태로 만든 후 DDM을 측정합니다.

만약 시스템 페이징이 정상이라면, 90° 케이블을 끼우든 안 끼우든 같은 DDM 값이 나옵니다 (cos(±90°)=0이므로 SBO 기여가 0). 만약 다른 DDM 값이 나온다면 위상이 어긋난 것.

5-2. Quadrature Phasing 절차 (교재 14장)

1
PIR을 미들마커 지점, 활공각 아래에 설치
활공각 위로 정확히 0.5θ 정도 아래 지점에서 측정 (150Hz가 우세한 지점)
2
CSB Only 방사 — 기준 DDM 측정
장비 전원 OFF → 측파대 안테나로 가는 전송선 분리 → 송신기 단자에 50Ω 더미로드 연결 → 전원 ON → CSB만 방사된 상태의 DDM 값 기록 (이게 기준값)
3
90° 케이블 삽입
장비 OFF → 측파대 전송선에 반송파 주파수의 90°에 해당하는 길이의 케이블 삽입
→ 330MHz GP에서 90° 케이블 = 약 15cm (VF=0.66 케이블)
→ 측파대 신호의 페이서가 시계방향 90° 회전한 효과
4
전원 ON → DDM 측정 → 송신기 위상 조정기로 조정
PIR의 DDM 수치가 단계 2에서 기록한 CSB Only DDM과 일치하도록 송신기의 SBO 위상 조정기를 천천히 돌림
→ 두 값이 정확히 같아지면 위상 조정 완료
5
90° 케이블 제거 → 확인 검사
장비 OFF → 90° 케이블 제거 → 전송선 복원 → 전원 ON
→ PIR의 DDM이 150Hz 우세를 정상적으로 표시하는지 확인

5-3. CEGP에서의 추가 고려사항

⚠️ CEGP는 NRGP보다 복잡
교재 14장 절차는 기본적으로 NRGP용이지만 CEGP에도 유사한 절차로 적용됩니다. 다만 CEGP는:
• 3개 안테나의 신호 합성이라 코스 신호 위상뿐 아니라 클리어런스 신호의 위상도 같이 점검
• CLR과 코스 신호의 측파대 성분이 역위상 관계여야 활공각 위에서 정상 동작
• 교재 13장: "상단과 하단 안테나에서의 클리어런스 신호의 RF 위상은 코스 신호 측파대 성분의 위상이 정상적으로 조정되면 자동으로 동기됨"

5-4. 비행검사로 위상 검증 — Dephasing 점검

비행검사 중 위상이 정상인지 확인하려면 Dephasing 점검을 합니다.

1
상단 안테나 전송선에 30° 위상 지연 삽입
동축 아답터(Bullet/Barrel)로 약 30° 위상 지연
2
비행검사 → 패스폭 측정
이때 패스폭이 어떻게 변하는지 기록
3
하단 안테나 전송선에 30° 위상 지연 삽입 (상단 제거)
반대 방향으로 같은 양 지연
4
비행검사 → 패스폭 재측정
이번에는 패스폭이 어떻게 변했는지 기록
5
두 패스폭 비교 → 위상 판정
• 두 패스폭이 거의 같다 (≤0.04° 차이): 위상 정상 ✓
• 차이가 있다: 위상 어긋남 → GP 위상 계산기로 조정량 계산하여 SBO 위상 보정
💡 위상 계산기 사용 원리
1. 신호의 특성곡선과 비행검사에서 측정한 두 패스폭들이 교차하는 그래프 지점에 표시
2. 수평축에 대한 점들로부터 수직으로 선을 그려 위상 불일치 각도 확인
3. 두 위상 불일치 각도들의 합 ÷ 2 = 상단 안테나의 위상 오차
4. 이 값만큼 SBO 측파대 신호의 위상을 보정 → 코스 폭의 양방향 변화량이 같아질 때까지 반복
06
클리어런스 (Clearance) 조정
"활공각 아래쪽 Fly-Up 신호가 약하다" → CLR 진폭 조정

6-1. 클리어런스 신호의 역할

CEGP 시설의 특징은 평탄지가 부족한 환경에서 사용된다는 점이에요. 그래서 지형 반사로부터 보호하기 위해 낮은 각도에서의 신호 방사를 줄이고, 그 대신 별도의 클리어런스 신호로 활공각 아래쪽 Fly-Up 신호를 보강합니다.

FORMULA — 클리어런스 신호 규정 (교재 13장)
E_CL = √(0.15 × E_c(MA)²) = 0.484
E_CL150 = 0.8 × E_CL = 0.387
0.387 = 상단과 하단 안테나로 공급되는 클리어런스 반송파 전력이 중단 안테나의 코스 반송파 전력의 15% 이하여야 한다는 FAA 규정.
→ 너무 크면 코스 신호의 Roughness(거칠음)에 영향 → 15% 이하 권장

6-2. 비행검사 항목 — Below Path / Above Path

항목측정 위치요구사항
Below Path
(아래쪽 장애물 회피)		 0.3θ ~ 0.45θ
(활공각의 30%~45%)		 150Hz 신호 충분 (Fly-Up 보장)
일반적으로 ≥190μA
Above Path
(하강지시 신호)		 활공각 위쪽 영역 90Hz 신호 우세 (Fly-Down 보장)
활공각 위 약 1.75θ까지

6-3. 클리어런스 진폭 조정

CLR 신호는 별도의 클리어런스 송신기에서 만들어져 상단·하단 안테나로 공급됩니다.

1
측정된 Below Path 값 확인
예: 1.5° 지점에서 측정 150μA → 기준 190μA보다 약함 → CLR 강화 필요
2
CLR 송신기 출력 조정
• 클리어런스 송신기 출력 단계적으로 증가 (보통 0.5~1dB 단위)
• 단, 코스 신호 전력의 15%를 초과하지 않도록 주의
• 너무 크면 Structure(코스 신호 거칠음) 항목이 나빠질 수 있음
3
Structure 항목 영향 확인
클리어런스 강화 후 비행검사에서 Glide Path Structure 항목도 재측정 — 활공경사상 ±30μA 이내 유지되는지 확인

6-4. 클리어런스 영향 트레이드오프

CLR 진폭과 검사 항목의 트레이드오프
07
변조도·균형 조정
"변조도가 안 맞는다" → 송신기 변조회로 조정

7-1. 두 가지 변조 관련 항목

📊변조도 (Modulation Depth)

90Hz·150Hz 각각의 변조 깊이.
• 운용개시: 40% ±2% (총 80%)
• 정기검사: 40% ±5% (총 75~85%)
→ 송신기 변조 회로 조정

⚖️변조 균형 (Symmetry)

90Hz와 150Hz의 비율 균형.
• CAT I: 33% ~ 67% (90Hz 변조도 기준)
• CAT II/III: 42% ~ 58% (더 엄격)
→ 변조 신호 진폭 밸런스 조정

7-2. 변조도가 어긋날 때

변조도 항목이 75% 미만 또는 85% 초과로 나타나면:

  • 총 변조도 조정: 송신기의 변조기(Modulator) 회로에서 90Hz·150Hz 신호의 진폭을 동일한 비율로 증감
  • 측정 위치: 변조모니터, 또는 PIR 직접 측정
  • 주의: 변조도가 너무 높으면 과변조(Over-modulation)로 신호 왜곡 → 측파대 비대칭

7-3. 변조 균형이 어긋날 때

대칭성이 깨지면 활공각에서도 DDM이 0이 아니게 됩니다. 90Hz와 150Hz 어느 한쪽이 더 강해진 거예요.

  • 주 조정: 송신기의 90Hz·150Hz 변조 신호 발생기에서 두 신호의 상대적 진폭 미세 조정
  • 변조 균형 측정: 활공각 정확히 위에서 DDM 측정. 0μA에 가깝게 (운용개시 ±5μA)
📌 변조 균형이 어긋난 경우 영향
변조 균형이 어긋나면 활공각 위치 자체가 이동한 것처럼 보입니다 (DDM=0인 지점이 실제 활공각이 아닌 곳에 형성). 이때 안테나 높이는 정상인데 활공각 항목이 안 맞을 수 있어요. 그래서 변조도·균형 → 활공각 → 패스폭 순서로 점검합니다.

8-2. 복합 증상 — 여러 항목이 동시에 안 나올 때

실제 비행검사에서는 한 가지 항목만 안 나오는 경우보다 여러 항목이 동시에 어긋난 경우가 많습니다. 이때는 우선순위를 따져야 해요.

🔄 조정 우선순위 (일반 원칙)
1. 변조도·균형 → 가장 먼저 (다른 측정의 기준이 됨)
2. 활공각 → 안테나 높이로 조정 (변조 균형이 맞은 다음)
3. 위상 → Quadrature Phasing (활공각이 맞은 다음)
4. 패스폭 → SBO 진폭 조정
5. 클리어런스 → CLR 진폭 조정 (코스 신호 영향 확인)
6. Structure → 환경적 요인 우선 점검, 필요시 CLR 미세 조정

이 순서를 지키는 이유: 앞 단계가 어긋난 상태에서 뒷 단계 조정은 잘못된 기준에서의 조정이 되어 오히려 악화될 수 있음.

9-3. 연습 시나리오 5개

Q1. 활공각 + 패스폭 동시 이상
비행검사 결과: 활공각 3.20° (목표 3.0°, 한계 ±0.15°), 패스폭 1.15° (목표 1.4° ±0.21°). 둘 다 한계 초과. 조정 순서와 방법은?
분석:
• 활공각이 +0.20° 높음 → 안테나가 의도보다 낮은 상태 (높아질수록 활공각 ↓)
• 패스폭이 0.25° 좁음 → SBO가 너무 강하거나 안테나 위치 이상

조정 순서:
1. 변조도·균형 먼저 확인 (이상 없으면 다음 단계)
2. 안테나 높이 먼저 조정:
   Δh ≈ 4.34m × (0.20/3.0) ≈ +29cm (안테나 올림)
   Lower +29cm, Middle +58cm, Upper +87cm
3. 안테나 조정 후 패스폭 재측정 — 안테나 올렸으면 패스폭이 자연스럽게 변함
4. 그래도 패스폭이 안 맞으면 SBO 진폭 미세 조정
5. 위상 점검(Quadrature) 후 클리어런스 재확인
6. 비행검사 재요청
Q2. 위상 비대칭 진단
Dephasing 점검에서 상단 +30° 시 패스폭 1.85°, 하단 +30° 시 패스폭 1.05°. SBO 위상은 얼마나 어긋나 있나? 어떻게 조정해야 하나?
분석:
• 두 패스폭이 차이 0.80° → 명확한 위상 어긋남 (정상이면 0.04° 이내)
• 상단 지연 시 패스폭이 넓어지는 것은 SBO 위상이 진상(Lead)임을 시사

조정 방법:
1. GP 위상 계산기 사용
   신호 특성곡선과 측정 패스폭 교점에 점 표시
   위상 불일치 각도 추정 → 합 ÷ 2 = SBO 위상 오차
2. 이 결과대로 송신기 SBO 위상 조정기를 보정
3. 보정 후 Dephasing 재실시 → 두 패스폭 차이 ≤0.04° 확인
4. Quadrature Phasing으로 최종 검증

주의: 위상 조정 후 활공각·패스폭 절대값도 약간 변함 → 비행검사 모든 항목 재확인 필요
Q3. Structure 항목 악화
3개월 전 정기검사에서 정상이었던 시설이 이번 검사에서 Zone 2 Structure ±35μA (한계 ±30μA)로 한계 초과. 같은 시기에 공항 인근 부지에 신축 빌딩이 들어섰다. 어떻게 접근할까?
분석:
• 시설 자체는 3개월 전 정상이었으니 GP 회로 이상 가능성 낮음
• 신축 빌딩 → 새로운 반사면 등장 → 다중경로 간섭 증가
• CEGP는 원래 이런 환경 보완용이지만 큰 구조물 등장 시 한계 초과 가능

접근 순서:
1. 환경 변화 분석: 신축 빌딩 위치·높이·반사 가능성 측정
2. 빌딩에서 오는 반사파가 활공경사 어느 지점에 간섭하는지 확인
3. CLR 진폭 미세 조정 시도 (1~2dB 감소 → 반사 영향 줄이기)
4. 안 되면 안테나 마스트 옵셋·방향 조정 검토
5. 그래도 해결 안 되면 다른 GP 형식(M-Array 등) 또는 운용 카테고리 조정 검토
6. 항공정보 발행 및 운영 부서 협의 (보호구역 확장 등)
Q4. 클리어런스 부족
활공각 1.4° 지점에서 측정 110μA (목표 ≥190μA). CLR을 강화해야 하는데 동시에 Structure 항목이 ±28μA (한계 ±30μA 직전). 어떻게 할까?
딜레마 분석:
• CLR을 키워야 Below Path 클리어런스 확보됨
• 그런데 CLR을 키우면 Structure가 한계 초과 위험
• 이미 ±28μA로 한계 ±30μA에 근접

접근 방법:
1. 먼저 위상 점검: Quadrature Phasing 정확도 확인 — 위상이 정확하면 Structure 자연스럽게 개선됨
2. CLR 진폭 측정값과 코스 신호 대비 비율 확인 (15% 이하인지)
3. CLR 0.5dB 단계로 미세 증가 → 각 단계마다 Below Path와 Structure 모두 측정
4. Below Path가 190μA에 도달하기 직전까지 단계적 조정
5. 만약 어느 단계에서 Structure ±30μA 도달 시 그 단계에서 중단 → CLR 진폭 한계 발견
6. 한계에 도달했는데도 Below Path 부족하면 환경 요인 점검(반사면) 또는 안테나 자체 점검
Q5. 종합 — 운용개시 검사 실패
신설 CEGP 시설 운용개시 검사 결과: 활공각 정상, 패스폭 1.6° (운용개시 한계 1.4°±0.07°), 변조 균형 −8μA (한계 ±5μA), Structure 정상. 어떤 순서로 조정할까?
먼저 분석:
• 변조 균형이 운용개시 한계 초과 (−8μA) — 최우선 조정 대상
• 패스폭도 운용개시 허용 ±0.07° 초과 (1.6°는 +0.2° 초과)
• 활공각·Structure는 정상이라 가산점

조정 순서:
Step 1: 변조 균형 조정
   송신기 90Hz/150Hz 변조 진폭 미세 조정 → ±5μA 이내로 (가능한 0μA에 가깝게)
Step 2: 활공각 재측정
   변조 균형 조정 후 활공각 위치가 약간 변할 수 있음 → 안 맞으면 안테나 높이 미세 조정
Step 3: 위상 점검 (Quadrature)
   변조 균형이 맞은 상태에서 SBO 위상 정확도 확인
Step 4: 패스폭 조정
   SBO 진폭 증가: 20×log(1.4/1.6) = −1.16dB → SBO를 약 1dB 강화
Step 5: 클리어런스·Structure 재확인
   변조 균형이 정확하고 SBO를 키웠으면 Structure가 약간 영향 받을 수 있음 → 확인
Step 6: 비행검사 재요청
   모든 운용개시 한계 충족 확인
