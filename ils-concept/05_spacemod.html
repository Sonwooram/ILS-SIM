01
공간변조(Space Modulation)란?
송신기가 아닌 공간에서 변조가 완성된다
💡 정의
반송파 신호(CSB)와 측파대 신호(SBO)를 별도의 안테나에서 분리 방사하고, 수신기 위치에서 두 신호가 공간에서 합성되어 AM 신호를 형성하는 기술. 수신기 위치에 따라 합성 위상이 달라져 변조도가 달라진다.
일반 AM (단일 안테나)

반송파+측파대가 한 안테나에서 방사 → 위상관계 고정 → 모든 방향 동일한 변조도

ILS 공간변조 (분리 방사)

CSB와 SBO가 다른 안테나에서 방사 → 수신 위치에 따라 위상차 변화 → 위치별 다른 변조도 → DDM 발생!

02
위상차(φ)의 역할
두 신호의 위상 관계가 변조도를 결정한다

CSB와 SBO가 수신기에 도달할 때의 상대 위상차 φ가 유효 변조도를 결정한다.

위상차 φ에 따른 합성 결과
03
유효 공간변조지수 Sf
위상차 φ가 변조도에 미치는 영향 — cosφ가 핵심
유효 공간변조지수
S_f = (E_ssm × cosφ) / E_cm
E_ssm = 수신된 분리 방사 측파대 신호 진폭
E_cm = 수신된 반송파 신호 진폭
φ = CSB와 SBO 사이의 위상차
cosφ가 핵심 — 위상차가 커질수록 유효 변조도 감소
위상차 φcosφS_fILS 의미
0°1.0최대완전 동위상 — 정상
45°0.70770.7%변조 감소
90°00DDM=0, 코스폭 무한대!
180°-1.0최대(역)역위상 — 송신기 변조와 감산
04
총 변조지수 — 송신기 변조 + 공간변조
두 변조가 합산되거나 상쇄된다
총 변조지수 (동위상)
M = m + S_f
총 변조지수 (역위상)
M = m − S_f
m = 송신기 변조지수
S_f = 유효 공간변조지수
동위상: 두 변조 합산 → 총 변조 증가
역위상: 두 변조 상쇄 → 총 변조 감소
⚡ 공간변조의 핵심 원리
LOC에서 SBO 패턴의 좌측 로브는 CSB와 동위상 → 90Hz 변조 합산(M 증가)
SBO 패턴의 우측 로브는 CSB와 역위상 → 90Hz 변조 감산(M 감소)
→ 좌측: 90Hz 우세 / 우측: 150Hz 우세 → DDM 발생!
05
위치별 신호 우세도
LOC 공간에서 항공기 위치에 따라 달라지는 신호
LOC 공간변조 — 위치별 90Hz/150Hz 우세도
💡 항공기 수신기의 판단
  • DDM = 0: 중심선 위 → 온 코스/온 패스
  • DDM > 0 (90Hz 우세): 오른쪽 벗어남 (LOC) 또는 패스 위 (GP)
  • DDM < 0 (150Hz 우세): 왼쪽 벗어남 (LOC) 또는 패스 아래 (GP)
06
연습문제
버튼을 눌러 정답 확인
📝 연습문제
Q1. CSB 100V, SBO 25V, 위상차 60°일 때 유효 공간변조지수 Sf를 구하라.
Sf = (E_ssm × cosφ) / E_cm = (25 × cos60°) / 100 = (25 × 0.5) / 100 = 12.5/100 = 0.125 (12.5%)
Q2. 송신기 변조지수 m=0.2, 공간변조지수 Sf=0.15(동위상)일 때 총 변조지수는? 역위상이면?
동위상: M = m + Sf = 0.2 + 0.15 = 0.35 (35%)
역위상: M = m − Sf = 0.2 − 0.15 = 0.05 (5%)
동위상 위치는 변조도가 크고, 역위상 위치는 변조도가 작다 → 이 차이가 DDM을 만든다.
