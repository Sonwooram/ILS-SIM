01
전자기파란 무엇인가?
안테나에 전류를 흘리면 왜 공간으로 에너지가 퍼져나가는가
⚡ 핵심 질문 먼저
전선에 전류를 흘리면 주변에 자기장이 생긴다 — 그런데 어떻게 그 에너지가 수십km 떨어진 항공기 안테나까지 도달할 수 있을까?

1-1. 발생 원리 — 연쇄 반응

도체에 교류 전류가 흐르면 주변 공간에 변화하는 자기장(H)이 생긴다. 변화하는 자기장은 주변에 전기장(E)을 만들고, 이 전기장은 다시 새로운 자기장을 만드는 연쇄 반응이 계속된다. 이렇게 서로를 만들어내며 공간을 달려나가는 것이 전자기파다.

전자기파 발생 연쇄 과정
💡 핵심 사실 3가지
  • 속도: 빛의 속도로 전파 — 약 3 × 10⁸ m/s (진공 중)
  • 매질 불필요: 공기도, 진공도 상관없이 전파 가능
  • 에너지 운반: 전기장과 자기장이 에너지를 실어 나름

1-2. 연못의 물결과 같은 원리

연못에 돌을 던지면 물결이 퍼져 나가듯, 안테나의 전류 변화가 공간에 전자기 물결을 만든다. 돌이 물에서 빠져나온 뒤에도 물결이 계속 퍼지듯, 전류를 끊어도 이미 공간에 나간 전자기파는 계속 달려간다.

02
전자기장의 두 종류
유도전자계 vs 복사전자계 — ILS에서 중요한 것은 어느 쪽인가

안테나 주변의 전자기장은 거리에 따라 성격이 완전히 다른 두 영역으로 나뉜다.

🔗 유도전자계 (Induction Field)
  • 안테나 근거리에서만 존재
  • 세기 ∝ 1/거리² — 멀어지면 급격히 약해짐
  • 전기장과 자기장이 90° 위상차
  • 에너지가 안테나·공간 사이를 왔다갔다 (저장·방출 반복)
  • 실제 전력 전달 없음 — 리액턴스 성분만
  • 통신에 사용 불가 (거리가 늘면 신호 없음)
📡 복사전자계 (Radiation Field)
  • 안테나에서 분리되어 공간으로 독립 전파
  • 세기 ∝ 1/거리 — 멀어져도 완만하게 감소
  • 전기장과 자기장이 동위상
  • 에너지가 안테나로 돌아오지 않고 계속 진행
  • 실제 전력 전달 — 저항성 성분
  • 통신·ILS에 사용되는 바로 이것!
거리에 따른 유도전자계 vs 복사전자계 세기 비교
📌 자유공간 임피던스 377Ω

복사전자계에서 전기장(E, V/m)과 자기장(H, A/m)의 비는 항상 377Ω으로 일정하다. 순수 저항이므로 전기장과 자기장은 항상 동위상 — 에너지가 한 방향으로만 흐른다는 뜻이다.

자유공간 임피던스
Z₀ = E / H = 377 Ω
03
전기장(E)과 자기장(H)의 관계
동위상, 직각 방향 — 전자기파의 3D 구조

복사전자계에서 전기장과 자기장은 세 가지 규칙을 항상 지킨다.

① 동위상

E와 H가 동시에 최대, 동시에 0이 된다. 에너지가 항상 양수 → 앞으로만 진행

② 서로 직각

E장 방향과 H장 방향이 90° 를 이룬다. 두 장이 3차원에서 서로 수직

③ 진행 방향과 직각

E와 H 모두 전파 진행 방향에 수직인 횡파(Transverse Wave)

전자기파의 3D 구조 — E장, H장, 진행방향이 서로 직각
💡 에너지 계산

전자기파 에너지의 절반은 전기장, 나머지 절반은 자기장이 담당한다.

복합 에너지 밀도 단위: W/m² (단위 면적당 전력)

04
파장과 주파수
f × λ = c — ILS 주파수 대역에서 파장이 얼마인가

4-1. 기본 관계식

파장 · 주파수 · 전파속도
λ = c / f
λ (lambda) = 파장 (m) — 파형 한 주기의 공간 길이
c = 빛의 속도 = 3 × 10⁸ m/s
f = 주파수 (Hz) — 1초에 반복되는 횟수
주파수↑ → 파장↓, 주파수↓ → 파장↑

4-2. ILS 주파수 대역별 파장

시설주파수 대역파장 (계산)의미
LOC (로컬라이저) 108 ~ 112 MHz λ ≈ 2.68 ~ 2.78m 반파장 안테나 ≈ 1.35m
GP (글라이드패스) 329 ~ 335 MHz λ ≈ 0.90 ~ 0.91m LOC 주파수와 쌍으로 배정
MB (마커비콘) 75 MHz λ = 4.0m 상향 방사로 항공기 통과 감지
DME 962 ~ 1213 MHz λ ≈ 0.25 ~ 0.31m 거리 측정용 UHF 대역
LOC 110MHz 파장 계산 예시
λ = 3×10⁸ / 110×10⁶ ≈ 2.73 m
→ 안테나에서 2.73m 이동할 때마다 위상이 360° 지연
→ 반파장(1.36m) 이동하면 위상 180° 지연 (역위상)
→ 이것이 배열안테나에서 위상차를 만드는 기본 단위!

4-3. 전기각(Electrical Angle)으로 거리 표현

ILS 교재에서 거리를 전기각(°)으로 표현하는 이유가 바로 이것이다. 파장 하나 = 360°로 정규화하면 주파수가 달라져도 같은 수식을 쓸 수 있다.

물리적 거리 → 전기각 변환
전기각(°) = (거리 / λ) × 360°
예: LOC 110MHz(λ=2.73m)에서 1.36m 이동 → (1.36/2.73)×360° = 180°
→ 반파장 이동 = 위상 180° 지연 = 역위상
05
편파 (Polarization)
전기장이 어느 방향을 향하는가 — ILS에서 수평편파를 쓰는 이유

전기장(E) 페이서의 방향이 반사면(보통 지표면)에 대해 어떤 방향을 향하는지에 따라 편파가 결정된다.

↕ 수평편파 (Horizontal Polarization)

전기장이 지표면에 평행(↔) 방향

  • 지표면에서 반사 시 위상·진폭 변화 없음
  • 이미지 안테나 전류 = 실제 안테나와 역위상
  • LOC, GP 모두 수평편파 사용
  • 지면 반사를 이용한 GP Null Reference 구성 가능
↔ 수직편파 (Vertical Polarization)

전기장이 지표면에 수직(↕) 방향

  • 지표면에서 반사 시 위상 180° 반전
  • 이미지 안테나 전류 = 실제 안테나와 동위상
  • 반사에 의한 패턴 변형이 크다
⚠️ 수평편파가 ILS에 중요한 이유

GP(글라이드패스)는 지면 반사를 의도적으로 활용해서 정밀한 활공각 패턴을 만든다. 수평편파를 쓰면 반사 시 위상이 보존되어 계산이 단순해지고 패턴을 정확히 제어할 수 있다.

비행검사에서도 LOC의 수평편파 성분이 없는지 확인하는 항목이 있을 만큼 편파는 ILS 성능에 직접 영향을 준다.

06
방사패턴 (Radiation Pattern)
안테나가 어느 방향에 얼마나 강하게 방사하는지를 그린 지도

6-1. 등방성 안테나 — 기준점

실제 안테나를 분석하기 전에 등방성 안테나(Isotropic Antenna)를 기준으로 쓴다. 등방성 안테나는 모든 방향으로 동일한 세기로 방사하는 이론적 안테나 — 방사패턴이 완전한 구(sphere) 형태다.

왜 등방성 안테나를 쓰나?

어떤 안테나든 먼저 등방성으로 가정하고 계산한 다음, 실제 안테나의 방향성을 반영하면 계산이 훨씬 단순해진다. ILS 교재 전체가 이 방법으로 배열안테나 패턴을 계산한다.

6-2. 방사패턴 표현 방법

📊 직각좌표 (Rectangular Plot)

가로축 = 각도(°), 세로축 = 전계강도 크기

로브의 크기와 위치를 정확한 수치로 읽기 좋다.

🎯 극좌표 (Polar Plot)

중심에서 방향별로 뻗어나간 거리 = 신호 세기

방사패턴의 전체 형태를 한눈에 볼 수 있다. ILS 교재에서 주로 사용.

극좌표 방사패턴 예시 — 등방성(원형) vs 다이폴(8자형)

6-3. Null — 신호가 0인 지점

방사패턴에서 신호가 0이 되는 방향을 Null이라 한다. ILS에서 Null은 매우 중요한 의미를 가진다 — SBO 안테나의 정면 Null이 코스 중심선을 만드는 핵심 원리이기 때문이다.

💡 다음 개념과의 연결

이 페이지에서 배운 전자기파의 파장, 위상 지연, 방사패턴 개념이 다음 페이지 페이서·벡터 합성의 토대가 된다. 특히 "거리 = 전기각"의 개념은 안테나 배열 계산 전체에 걸쳐 핵심적으로 쓰인다.

08
연습문제
버튼을 눌러 정답을 확인하세요
📝 연습문제 — 개념 확인
Q1. 전자기파가 진공(매질 없음)에서도 전파될 수 있는 이유를 설명하라.
전기장과 자기장이 서로를 만들어내는 연쇄 반응으로 전파되기 때문이다. 변화하는 자기장이 전기장을 유도하고, 그 전기장이 다시 자기장을 만드는 과정이 반복되므로, 물질(매질)의 진동이 필요한 음파와 달리 진공에서도 전파된다.
Q2. 유도전자계와 복사전자계의 차이를 거리 감소율과 에너지 전달 측면에서 비교하라.
유도전자계는 세기가 1/거리²에 비례하여 급격히 감소하고, 에너지가 안테나와 공간 사이를 왔다갔다 하므로 실제 전력 전달이 없다(리액턴스 성분). 반면 복사전자계는 세기가 1/거리에 비례하여 완만히 감소하고, 에너지가 안테나로 돌아오지 않고 계속 전파되므로 원거리 통신에 사용된다. ILS는 복사전자계를 이용한다.
Q3. LOC 주파수 110.3MHz에서의 파장을 계산하고, 안테나 간격 1.5m가 몇 전기각(°)인지 구하라. (c = 3×10⁸ m/s)
파장: λ = 3×10⁸ / 110.3×10⁶ ≈ 2.72m

전기각: (1.5 / 2.72) × 360° ≈ 198.5°

→ 1.5m 간격은 약 198.5°의 위상 지연에 해당한다. 이 값이 안테나 배열에서 합성 패턴(방사패턴)의 형태를 결정하는 핵심 변수 'a'가 된다.
Q4. ILS LOC와 GP가 수평편파를 사용하는 이유를 지면 반사와 연관지어 설명하라.
수평편파로 방사된 전자기파가 지표면에서 반사될 때, 위상과 진폭이 변하지 않는다. 이는 이미지 안테나 전류가 실제 안테나와 동위상임을 의미하므로, 지면 반사파를 예측하고 활용하는 GP Null Reference 방식의 설계·분석이 단순해진다. 반대로 수평편파는 반사 시 180° 위상 반전이 생겨 패턴 계산이 복잡해지고 의도치 않은 신호 왜곡이 발생할 수 있다.
Q5. 자유공간 임피던스가 377Ω(순수 저항)이라는 사실이 전기장과 자기장의 위상 관계에 대해 무엇을 의미하는가?
순수 저항 회로에서는 전압과 전류가 동위상이다. 자유공간의 임피던스가 저항성(377Ω)이라는 것은, 이와 유사하게 전기장(E)과 자기장(H)이 항상 동위상임을 의미한다. 두 장이 동위상이면 순시 전력(E×H)이 항상 양수 — 즉, 에너지가 항상 앞방향으로만 흐르고, 안테나로 역행하지 않는다. 이것이 복사전자계가 에너지를 실어 나를 수 있는 이유다.
