01
DDM이란? (Difference in Depth of Modulation)
항공기 위치를 하나의 숫자로 표현하는 ILS의 핵심 지표
⚡ DDM의 의미
항공기 수신기가 90Hz 변조도와 150Hz 변조도를 측정하고, 그 차이를 계산한다. 이 차이가 항공기가 코스/패스에서 얼마나, 어느 쪽으로 벗어났는지를 알려준다.
DDM 정의식
DDM = m₉₀ − m₁₅₀
m₉₀ = 90Hz 변조도 (소수 표현, 예: 0.20 = 20%)
m₁₅₀ = 150Hz 변조도
DDM은 스칼라량 — 위상에 무관하게 진폭 크기만 비교
범위: 0(온 코스) ~ 최대 1.0
02
DDM 계산
측정값에서 DDM 구하기
DDM 계산 예시
DDM = m₉₀ − m₁₅₀ = 0.24 − 0.16 = 0.08
예: 90Hz 변조도 24%, 150Hz 변조도 16% 측정 시
DDM = 0.24 − 0.16 = 0.08 → 90Hz 우세 → 코스 오른쪽에 위치

총 변조도 합: m₉₀ + m₁₅₀ = 0.24 + 0.16 = 0.40 (일정)
온 코스가 아닌 위치에서도 합은 일정하고, 두 값의 분배만 달라진다.

📌 계기 지침과 DDM의 관계
항공기 CDI(Course Deviation Indicator) 지침의 최대 편향 = DDM 0.155 (LOC) 또는 0.175 (GP)
DDM이 이 값에 도달하면 지침이 끝까지 치우친 것 → 이 이상 벗어나면 코스에서 완전히 이탈
04
LOC 코스폭 (Course Width)
활주로 길이에 따라 결정되는 코스각도

LOC 코스폭은 활주로 시단(Threshold)에서 ±350ft (약 107m) 위치가 DDM=0.155가 되도록 설정한다.

코스폭 계산
θ = arctan(350 / D) × 2
D = LOC 안테나에서 활주로 시단까지 거리 (ft)
D = 활주로 길이 + 1000ft (LOC 안테나 위치 고려)
예: 활주로 9,000ft → D = 10,000ft → θ = arctan(350/10000)×2 = 2.0°×2 = 4.0°
활주로 길이D (ft)코스폭범위
짧은 활주로~5,000ft~8°3°~6° 이내 조정
표준 활주로 (8,000ft)9,000ft~4.4°정상 범위
긴 활주로 (12,000ft)13,000ft~3.1°정상 범위
05
GP 패스폭 (Path Width)
활공각 위아래 DDM=0.175 구간

GP 패스폭은 활공각(보통 3.0°)을 중심으로 위아래 DDM=0.175인 지점까지의 각도 범위다.

GP 패스폭 (Path Width)
PW = 0.24 / A    (도 단위, 약식)
A 비율 = E_ss/E_cs = (패스폭/2 + 활공각) / 10 (약식)
정상 패스폭: 1.4° (±0.7°)
패스폭이 너무 좁으면: 항공기가 패스에서 쉽게 이탈
패스폭이 너무 넓으면: 착륙 강하각 정밀도 저하
📌 LOC vs GP DDM 기준 비교
  • LOC Full Scale: DDM 0.155 → 코스폭 3°~6°
  • GP Full Scale: DDM 0.175 → 패스폭 1.4°
  • GP 기준값이 큰 이유: 수직면 정밀도가 더 중요(착륙 강하각), 패스폭이 좁아 같은 이탈 각도에서 DDM이 더 크게 변해야 함
06
DDM 곡선과 선형성
코스/패스 근처에서 DDM은 선형적으로 변해야 한다

정상적인 ILS에서 DDM은 코스/패스 중심에서 약 ±1° 이내에서 선형적으로 변화해야 한다. 비선형 구간에서는 항공기 오토파일럿 추적 성능이 저하된다.

07
연습문제
버튼을 눌러 정답 확인
📝 연습문제
Q1. 측정 결과 90Hz 변조도 22%, 150Hz 변조도 18%일 때 DDM과 항공기 위치를 판정하라 (LOC 기준).
DDM = 0.22 − 0.18 = 0.04
DDM > 0 (90Hz 우세) → 항공기가 코스 오른쪽에 위치. DDM 0.04는 Full Scale(0.155)의 약 26% 지점 → 코스에서 약 1/4 이탈
Q2. 활주로 길이 10,000ft인 공항의 LOC 코스폭을 계산하라.
D = 10,000 + 1,000 = 11,000ft
θ = arctan(350/11,000) = arctan(0.0318) ≈ 1.82°
코스폭 = 1.82° × 2 = 약 3.6°
(ICAO 허용 3°~6° 이내로 정상)
Q3. GP DDM Full Scale이 LOC(0.155)보다 높은 0.175인 이유를 설명하라.
GP 패스폭(1.4°)이 LOC 코스폭(3°~6°)보다 훨씬 좁다. 같은 이탈 각도에서 더 민감하게 반응해야 하기 때문에 Full Scale DDM 기준을 높게(0.175) 설정하여 패스 이탈 감지 감도를 높인다. 또한 착륙 강하 단계에서 수직 정밀도가 수평 정밀도보다 더 중요하기 때문이다.
